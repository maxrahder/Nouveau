Now we want to be able to look at the fields of the model record in the detail pane on the right (east).

  1. From the Toolbox, drag 5 Display Fields onto the Detail Form in the Project Inspector.

  2. Select each of the fields, one after the other, in the Config panel, give them the following property values:

    • name: name, fieldLabel: Name

    • name: artist, fieldLabel: Artist

    • name: link, fieldLabel: Link

    • name: category, fieldLabel: Category

    • name: price, fieldLabel: Price

    Note: If you click on the lock to the right of the Config filter area, it will keep the property configuration selected and you will be able to select each display field and change that property below.

  3. Each of these fields should have an anchor value of 100%.

    Note: Your code should look like this:

    {
    xtype: 'form',
    flex: 1,
    region: 'east',
    bodyPadding: 10,
    title: 'Detail Form',
    items: [
        {
            xtype: 'displayfield',
            anchor: '100%',
            name: 'name',
            value: 'Display Field',
            fieldLabel: 'Name'
        },
        {
            xtype: 'displayfield',
            anchor: '100%',
            name: 'artist',
            value: 'Display Field',
            fieldLabel: 'Artist'
        },
        {
            xtype: 'displayfield',
            anchor: '100%',
            name: 'link',
            value: 'Display Field',
            fieldLabel: 'Link'
        },
        {
            xtype: 'displayfield',
            anchor: '100%',
            name: 'category',
            value: 'Display Field',
            fieldLabel: 'Category'
        },
        {
            xtype: 'displayfield',
            anchor: '100%',
            name: 'price',
            value: 'Display Field',
            fieldLabel: 'Price'
        }
    ]
}
