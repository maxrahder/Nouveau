Sencha Desktop Packager is an easy-to-use, command-line tool that enables you to package web applications based on Chromium — written with web technologies such as HTML5, CSS3 and JavaScript — as native desktop applications. Now you can deploy your web applications as high-performance native apps for Windows, Mac OS X and Linux systems without dependency on the browser. Desktop Packager provides a wide variety of device APIs that enable access to the file system, menus, icons, and more.
