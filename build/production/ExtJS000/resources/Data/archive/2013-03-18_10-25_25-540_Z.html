Now that you have built your theme, you need to switch your application to the new stylesheet.

You can change this by editing the .sencha/app/sencha.cfg file.

//app.theme=ext-theme-classic
app.theme=ext-theme-mycustomtheme

After switching the theme. You will need to build the app: 

sencha app build

Note: .sencha is a hidden folder, to edit this you need to show hidden files in Explorer / Finder. See appendix for instructions...
