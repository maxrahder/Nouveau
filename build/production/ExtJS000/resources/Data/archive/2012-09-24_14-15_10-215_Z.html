Sencha GXT is the fastest, most powerful way to create rich web-based applications using Java. Sencha GXT uses the Google Web Toolkit (GWT) compiler that allows you to write your applications in Java and compile your code into highly optimized cross-browser HTML5 and JavaScript. Sencha GXT takes GWT to the next level, giving you high-performance widgets, feature-rich templates and layouts, advanced charting, data loaders and stores, and much more.
