The following Ext components have a ui-mixin to create a new ui:

  • Ext.panel.Panel
  • Ext.toolbar.Toolbar
  • Ext.window.Window
  • Ext.button.Button

Each of these mixins is named as: extjs-componentname-ui.
You can find it in the API docs, which CSS Mixins are applied.
For example: Toolbar

Note: While UI mixins are a handy way to configure multiple appearances for a component, they should not be overused. Because each call to a UI mixin generates additional CSS rules, gratuitous calls to UI mixins can produce an overly large CSS file.
