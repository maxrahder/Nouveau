Sencha Cmd is a nice tool on the command line for scaffolding, minifying and deploying your application to production.

It can generate:

  • applications
  • themes
  • models
  • forms
  • controllers
  • packages

Although it's handy to generate code with Sencha Cmd, it's not required.

However, if you are planning to create deployment packages with it, you should use Sencha Cmd from the beginning.
