Introduction

This lab will let you play with Ext.XTemplate and Ext.view.View. You will create a little movie application that loads the most populair movies (from iTunes) in your app. When you hover over each item it will display a rollover style. When you click on an item it opens the movie trailer in YouTube.

Note: You will need the following file: Labs/Templates/app.js

Steps

  1. Make sure the data loads from the Store instead of the array.

    As you can see the the data is not dynamic, we will use a proxy on the Ext.data.Model to retrieve data from iTunes. Uncomment the model and store and delete the data array. You can also remove the movieTpl.overwrite() indicated by the comment.

  2. Check the Ext.window.Window code.

    Right now the child item has no xtype defined so it will be automatically configured to its default xtype: panel.

    Use the xtype: dataview. It should be connected to the Movies store: Ext.data.StoreManager.lookup('Movies'), and it should use the movieTpl template.

  3. Set the itemSelector to div.holder

    Now test your code in the browser. You should see... ...actually you will see almost same as when you started this lab. Except that you will see more movies loaded from iTunes.

    The most power in this dataview is in its listeners. You will start coding 2 listeners. The itemclick, this handler opens the movie in YouTube and the itemmouseenter. This handler shows a nice mouse rollover.

  4. Create both listeners
  5. itemclick will contain the following function: 
    fn: function(e, target){
   window.open('http://www.youtube.com/results?search_query=' + target.data.movie + '+trailer', '_blank');
}
  6. itemmouseenter will contain the following function: 
    fn: function(e, target, item){
    Ext.each(Ext.query('.hover'), function(){
       this.style.display = 'none';
    });
    Ext.query('.hover', item)[0].style.display = 'block';
}

Test your application in the browser.
