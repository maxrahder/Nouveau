1. From the Toolbox, place a simple View inside the MainPanel.

    It should look like this:

  2. In the Config panel give this new view the following property value:

    1. autoScroll: true (checked)

  3. Select the [Code] button.

    4. Note: Architect has automatically created the following properties for this view, in addition to the autoScroll value you have just selected.

       items: [             
       {
            xtype: 'dataview',
            region: 'center',
            autoScroll: true,
            itemSelector: 'div',
            itemTpl: [
               'Data View Item {string}'
            ]
       } 
   ]

    Note: We will come back to the itemSelector and itemTpl a little later.

    Save the Application
