Setting the Locale of your Application Statically

Locales are implemented as JavaScript files that are located in your /extjs4/locale folder. The filename indicates the locale, i.e. ext-lang-fr.js is the localization file for French (France). You can change the locale of your application by simply linking to one of these files as illustrated by the following code snippet:

<!DOCTYPE html>
<html>
<head>
    <!-- Ensure we're using UTF-8 -->
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
    <title>Localization example</title>
    <!-- Main Ext JS files -->
    <link rel="stylesheet" type="text/css" href="extjs/resources/css/ext-all.css">
    <script type="text/javascript" src="extjs/ext-debug.js"></script>
    <!-- Include the translations -->
    <script type="text/javascript" src="extjs/locale/ext-lang-fr.js"></script>

    <script type="text/javascript" src="app.js"></script>
</head>


Note: You will need to refresh the entire page when switching locales. Ext JS does not support changing the locale on the fly (like updating the texts inside already rendered components)..
