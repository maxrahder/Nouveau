Another way of sending requests and receiving responses is JSON with Padding (JSON-P).

JSON-P is a way of processing data retrieved from a different domain than the one serving the page. In other words, it's a work-around for the "same origin policy" built into browsers. 

Ext.Ajax.request({
    url: 'https://itunes.apple.com/us/rss/topmusicvideos/limit=100/json',
    failure: function(response) {
        // Note network traffic -- the browser sends an OPTIONS request, and the response is a 501 (not implemented)
        alert('No can do.');  
    }
});
Ext.data.JsonP.request({
    url: 'https://itunes.apple.com/us/rss/topmusicvideos/limit=100/json',
    success: function(response) {
        alert('We got data!');
        console.log(response.results);
    }
});
