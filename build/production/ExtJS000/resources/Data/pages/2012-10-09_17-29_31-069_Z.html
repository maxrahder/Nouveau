It's so likely you'll dock toolbars, Ext JS has convenience properties for specifying a top-, bottom-, left- and right-docked toolbar: tbar, bbar, lbar and rbar. 

Ext.create('Ext.panel.Panel', {
    title : 'My Panel',
    height : 200,
    width : 340,
    renderTo : Ext.getBody(),

    tbar : [ {
        text : 'Button'
    }, {
        text : 'Button'
    } ]

});
