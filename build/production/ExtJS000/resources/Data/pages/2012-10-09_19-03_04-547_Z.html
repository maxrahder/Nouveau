There are some components designed for use within toolbars:

Ext.toolbar.TextItem xtype:'tbtext'
Ext.toolbar.Spacer xtype:'tbspacer'
Ext.toolbar.Separator xtype:'tbseparator'
Ext.toolbar.Fill xtype:'tbfill'

Each of these is a component, so you can specify component configs like style, height or width. Each has an xtype.
