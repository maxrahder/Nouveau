There are several places to find out more about Sencha Cmd. A handy way to find out what your options are is to simply use the command line. As an example, type Sencha app or Sencha generate from the command line for a list of the commands and options available for a specific category. See the guides in the Sencha Ext JS documentation for more information on Sencha Cmd. Also see the Sencha Cmd Reference for a summary of all command syntax.
