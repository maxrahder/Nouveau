Note the requires array. A requires array is needed for all objects created via their alias. We'll talk about requires in more detail later. 

Ext.define('MyApp.view.Viewport', {
    extend : 'Ext.container.Viewport',    
    requires : [ 'MyApp.view.BeatlesGrid' ],
    layout : 'fit',
    items : [ {
        xtype : 'beatlesgrid',
        store : 'People'
    } ]
});
