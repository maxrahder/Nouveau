Introduction

Let's start with creating your first full working MVC application. Does this sound scary? You don't need to be. Since you will generate this application it requires less coding skills. This lab requires that the latest version of Sencha Cmd is correctly installed.

You will create the MyDemoApp in the XAMPP/htdocs/Student/labs folder.

Steps

  1. Verify that Sencha Cmd is correctly installed and added to the class path.

    Open your console/terminal and enter the command: sencha which. It should prompt you the Sencha Cmd version number. If the version number < 3.0.2, reinstall Sencha Cmd, please follow the install guide. If you are having problems, please ask your instructor to help you.

  2. Prepare the folder structure

    In the htdocs/Student folder, create a folder called: labs (htdocs/Student/labs).

    In the htdocs folder, make sure there is a copy of Ext JS in a folder named: extjs4.

  3. Navigate with your console/terminal navigate to the extjs4 folder.
  4. Generate your app by entering the following command: 
    sencha generate app MyDemoApp ../Student/labs/MyDemoApp

    (You can almost generate everything with Sencha Cmd; type in your console: sencha help generate to see an overview of what else you can generate.)

  5. Open the MyDemoApp http://localhost/Student/labs/MyDemoApp/ in your browser.

    You should see your MyDemoApp application, with 2 columns. The center column contains a tabpanel.

  6. Review the generated folder structure of the MyDemoApp in Finder/File Explorer.

Keep the files on your localhost. You will use it later for the Eclipse Lab.
