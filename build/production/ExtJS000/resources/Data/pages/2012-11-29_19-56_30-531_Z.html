• icon — the path to an image
  • iconCls — a CSS class specifying a background image
  • iconAlign — 'top' | 'right' | 'bottom' | 'left'

See the documentation to see various size and icon combinations. 

Ext.create('Ext.panel.Panel', {
    title : 'Panel',
    renderTo : Ext.getBody(),
    height : 200,
    bodyPadding : 16,
    defaults : {
        margin: 8
    }, 
    layout : 'vbox',
    items : [{
        xtype : 'button',
        text : 'Sunny',
        icon : 'resources/images/weather_sun.png'
    }, {
        xtype : 'button',
        text : 'Rainy',
        icon : 'resources/images/weather_rain.png',
        iconAlign : 'right'
    }]
});
