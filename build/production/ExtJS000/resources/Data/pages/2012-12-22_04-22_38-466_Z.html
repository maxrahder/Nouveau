• An Ext JS class is a function.
> var c = Ext.define( 'Thing', {} ) // define the class
  undefined
> c // returns a function named 'constructor'
  function constructor() {
      //....
     return this.constructor.apply(this. arguments) || null;
  }
  • The Ext JS class-function is therefore an object. It has a name property.
> c.name // returns the name of the function in c
  "constructor"
  • The c class-function also has a $className property.
> c.$className
  "Thing"
