Call Ext.define() to define a class. Use Ext.create() to create an instance of a class.

The define() function takes two required parameters: the name of the class, and a config object that describes the new class. 

Ext.define('Mammal', {});
var m = Ext.create('Mammal');
alert('Instance of ' + m.$className + ' which extends ' + m.superclass.$className);

The $className property contains the class name. Knowing this can be handy when debugging.
