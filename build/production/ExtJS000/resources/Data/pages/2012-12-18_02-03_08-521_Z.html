idProperty property in a model:
  • points to the "primary key" in the field definition
  • update and delete operations use this property to locate the affected row 
Ext.define('MyApp.model.MyModel', {
    extend : 'Ext.data.Model',

    idProperty: 'id',

    fields : [
        { name: 'id' },
        { name: 'lastname', type: 'string' }
    ],

    validations: [ ... optional validations ... ],

    yourFunction: function() {
        // Your code
    }
});
