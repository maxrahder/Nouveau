Behind the scenes, the set of tabs is in a specially styled, north-docked container, Ext.tab.Bar.

Each tab is a specially styled button, Ext.tab.Tab.

Ext.create('Ext.tab.Panel', {
    renderTo : Ext.getBody(),
    height : 200,
    width : 200,    
    plain : true,
    items : [{
        title : 'One',
        html : 'Panel One'
    }, {
        title : 'Two'
    }]
});
