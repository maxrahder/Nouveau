• The child nodes may also have their own children.

  • The expanded property determines if the the branch will be opened initially. The default is false.

  • If a node has no children, if must have a leaf property with a value of true. Failing to set the leaf property to true can lead to unintended children.

Ext.create('Ext.tree.Panel', {
    root: {
        text: 'Root Node',
        expanded: true,
        children: [{
            text: 'Child 1 (expanded: true)',
            expanded: true,
            children: [{
                text: 'Child 1.1 (leaf: true)',
                leaf: true
            }, {
                text: 'Child 1.2 (leaf: false)',
                leaf: false
            }]
        }, {
            text: 'Child 2 (expanded: false)',
            expanded: false,
            children: [{
                text: 'Child 2.1 (leaf: true)',
                leaf: true
            }]
        }]
    },
    title: 'Tree With Root and Children',
    renderTo: Ext.getBody()
});
