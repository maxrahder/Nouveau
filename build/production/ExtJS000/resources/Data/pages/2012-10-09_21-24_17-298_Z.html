As you saw earlier, you can specify event handlers via the listeners config property. Since the "click" event is the most likely thing you need to listen to, buttons have a convenience property, handler for specifying the click event. 

Ext.create('Ext.panel.Panel', {          
	bodyPadding: 16,
	renderTo : Ext.getBody(),
	layout : 'vbox',
	items : [ {
		xtype : 'button',
		text : 'Button One',
		handler: function(b){
			alert('You clicked the ' 
			+ b.getText() 
			+ ' button.');
		}
	}]
});
    
Ext.create('Ext.panel.Panel', {          
	bodyPadding: 16,
	renderTo : Ext.getBody(),
	layout : 'vbox',
	items : [ {
		xtype : 'button',
		text : 'Button Two',
		listeners : {
			click: function(b){
			    alert('You clicked the ' 
			    + b.getText() 
			    + ' button.');
		    }
		}
	}]
});
