1. Have the data view show data

    We will now put images for each video in the lower panel.

    In the Project Inspector, choose neptunesView, and in the Design canvas flyout gear set store to Tunes.

    Data views render data using a template, which is a tokenized HTML snippet. In the Property Inspector panel double-click on itemTpl under tunesView.

    This opens the code window on the Canvas. Copy and paste the following snippet into the code window, replacing the default Data View Item {string} that's there. Note that the snippet below contains a token for {title} and {artist}. Those values from the store's records will be shown. 

    <figure>
    <img src="{image}" />
    <figcaption>{title}<br/>{artist}</figcaption>
</figure>
  2. Save and preview the application

    On the Architect main toolbar, press [Save] and then refresh your browser. You should see the NepTunes Grid populated and video thumbnails on the NepTunes Music Videos. Try sorting by Artist in the grid and watch the view update.

  3. Improve the thumbnail item layout

    The video thumbnails in the view are shown as a single column. These would look better with a little styling.

    In the Project Inspector, select the [+] button and select Resource | CSS Resource

    In the Project Inspector, select the new resource (under Application | Resources) and in the Config panel set url to resources/style.css

    Choose the Code View to see the new style sheet. It defines a video CSS class which is floating, with a specific height and width.

    In the Project Inspector, choose neptunesView and in the Config panel type Cls in the filter.

    Set itemCls to video.

    Set the overItemCls to overvideo.

    In the Config for neptunesView, in Filter type track, then set trackOver checkbox to checked.

  4. Save and preview the application

    On the Architect main toolbar, press [Save] and then refresh your browser. Note that the images are now in rows, and the effect you get when you hover the mouse pointer over an image.

  5. Create the preview window

    On the Toolbox select a Views | Window and drag it onto Views in the Project Inspector. It should be at the same indentation level as MyViewport.

    In the Project Inspector, select the new Window, and in the Config panel set its userClassName to PreviewWindow.

    In the Config panel filter type resizable: false to prevent window resizing.

    In the Config panel use Quickset in the Filter to set height: 352 and width: 640.

    In the Config panel, set the modal checkbox to checked.

    In the Config panel, find the tpl property, and press the (+) icon to the right to create a template for the window.

    In the Property Inspector, double-click on tpl under PreviewWindow to open the Code editor in the Canvas.

    Copy and paste the following as the template. Note that the template has a {preview} token, which corresponds to a field in the model.

    <video autoplay preload="auto">
  <source src="{preview}" type="video/mp4">
</video>
  6. Create the controller

    Press the [+] at the top right of the Project Inspector and choose Controller.

    In Project Inspector select the new controller and in the Config panel set userClassName to NepTunesController.

  7. Add control logic

    In the Project Controller, select neptunesView.

    In the Config panel, find the EventBindings item, and click on the (+) to the right.

    Select the itemdblclick event.

    To move the event listener to the controller, right-click on itemdblclick onNepTunesViewItemDblClick, select Convert To Action > NepTunesController.

    In the Project Inspector, double-click on NepTunesController to see the code generated.

    In the Project Inspector, double-click on itemdblclick onNepTunesViewItemDblClick method now under the NepTunesController, and copy and paste in the following code:

    var window = 
    Ext.create('NepTunes.view.PreviewWindow', 
                   {title: record.get('title') +
                    ', provided courtesy of iTunes'
                   }
              );
window.update(record.data);
window.show();

  8. Save and preview your application

    On the Architect main toolbar, press [Save] and then refresh your browser. You should see your completed application. It has a two panels labeled NepTunes Music Videos and NepTunes Grid. The NepTunes Grid shows a grid. The NepTunes Music Videos view shows video thumbnails. Double-clicking on an item in the Videos view brings up a window showing a video preview.
