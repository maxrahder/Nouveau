Introduction

In this lab, you will work with Sencha Architect. You will use Architect to create a Sencha Touch project and define Views for a Twitter search mobile application.

Steps

  1. Open Sencha Architect.

  2. Under the heading Create New Project, Select Sencha Touch 2.1.x (the latest supported version)

  3. After Sencha Architect opens, click the Save button on the toolbar and enter the following information:

    • Save Path: {path to web server root}/SenchaTraining/TouchTraining/Labs/Architect

    • Project Name: TwitterSearch

  4. Click the Save button.

  5. Click the Settings button on the Main toolbar.

  6. Click the Project tab and enter the following:

    • URL prefix: http://localhost/SenchaTraining/TouchTraining/Labs/Architect/TwitterSearch

  7. Click Save.

  8. Select Application in the Project Inspector.

  9. In the Config panel, start to type name into the filter field. This will filter to the name property.

  10. Set the name property value to TwitterSearch.

  11. In the Toolbox, make sure Everything is selected.

  12. In the Toolbox filter, type in tab.

  13. Drag a Tab Panel from the Toolbox and drop it onto the Design Canvas.

  14. In Project Inspector, under Views, click the MyTabPanel

  15. In the Config panel, start to type user into the filter field. This will filter the user named properties.

  16. Select userClassName below and set the value to TweetsTabPanel.

  17. In the Project Inspector, delete the Tab 1, Tab 2, and Tab 3 views by right-clicking on each and selecting Delete.

  18. In the Toolbox filter, type in list.

  19. Drag a List view from the Toolbox and drop it onto TweetsTabPanel in the Design Canvas.

  20. Double-click the MyList button and change to Tweets.  This changes the title config to Tweets.

  21. In the Toolbox filter, type in tool.

  22. Drag a Toolbar from the Toolbox and drop it onto the Tweets list view in the Design Canvas.

  23. In the Toolbox filter, type in search.

  24. Drag the Search Field from the Toolbox and drop it onto MyToolbar in the Design Canvas.

  25. Double-click the Search Field label and remove the Field text.

  26. In the Project Inspector, click the MySearchField.

  27. In the Config filter, type in place.

  28. Change the placeHolder value to Search....

  29. In the Config filter, type in flex.

  30. Change the flex value to 1.  This sets the width to proportionally fill its container.

  31. In the Toolbox filter, type in map.

  32. Drag the standard Map component from the Toolbox and drop it onto TweetsTabPanel in the Design Canvas. Note: Make sure you select the standard map component and not the Google Map API.

  33. Double-click the MyMap button and change to Map.

  34. In the Toolbox filter, type in form.

  35. Drag the Form Panel from the Toolbox and drop it onto TweetsTabPanel in the Design Canvas.

  36. Double-click the MyFormPanel button and change to Settings.

  37. Click the Settings button.

  38. In the Toolbox filter, type in text.

  39. Drag the Text Field from the Toolbox and drop it onto Settings in the Project Inspector.

  40. Double-click the Field label and change to Default Search.

  41. In the Project Inspector, click the MyTextField.

  42. In the Config filter, type in label.

  43. Change the labelWidth to 50%.

  44. Click Save.

  45. Click Preview.  If needed set the url: http://localhost/SenchaTraining/TouchTraining/Labs/Architect/TwitterSearch

Conclusion
