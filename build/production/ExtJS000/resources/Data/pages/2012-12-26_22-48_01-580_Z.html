Card layout can have multiple children, but only one is visible at a time. The visible child takes up all the available space.

Ext.layout.container.Card has three methods for changing the item being shown: setActiveItem(), next() and prev().

setActiveItem() takes one parameter, which can be the index of the item to be shown, a reference to the item itself or an item ID. 

Ext.create('Ext.panel.Panel', {

    layout : 'card',

    items : [ {
        title : 'Ready',
        html : 'Ready...'
    }, {
        title : 'Steady',
        html : 'Steady...'
    }, {
        title : 'GO!',
        html : 'GO!'
    } ],
    buttons : [ {
        text : '<',
        handler : function(b) {
            b.up('panel').getLayout().prev();
        }
    }, {
        text : '>',
        handler : function(b) {
            b.up('panel').getLayout().next();
        }
    } ],

    bodyPadding : 6,
    defaults : {
        bodyPadding : 16
    },
    height : 200,
    width : 200,
    renderTo : Ext.getBody()

});
