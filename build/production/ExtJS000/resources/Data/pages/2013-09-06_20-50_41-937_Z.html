The files in sass/var and sass/src must be named after your application's package/class names. Only matching files are processed.

For example, if your application has a file named app/view/Viewport.js, you can create a file named sass/var/Viewport.scss — it will be automatically processed when you build your application.

Place Sass code in the file that most closely relates to the corresponding affected component. Global variables and Sass source can be placed in Application.sass or Viewport.sass.
