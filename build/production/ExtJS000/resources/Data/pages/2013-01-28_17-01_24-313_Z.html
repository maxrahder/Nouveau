Sencha Cmd is a command-line tool that makes it quick and easy to do several application-development tasks.

  • Use sencha generate app to create an initial stubbed out applications
  • It's also possible to generate models, controllers and views
  • Use sencha fs web for using the built-in web server
  • Use sencha app build to create a minified library file containing only those components used in your application
  • There are also commands for theming, upgrading Ext JS versions, etc.

We will be using Sencha Cmd in several of the labs this week.
