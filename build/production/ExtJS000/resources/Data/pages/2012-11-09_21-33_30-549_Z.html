Ext.layout.container.Anchor

The default form panel layout is anchor. Child items are sized relative to relative to the container's dimensions. If the container is resized, all anchored items are automatically re-rendered according to their anchor rules. This is sometimes also referred to as a "liquid layout". 

Ext.create('Ext.form.Panel', {
    title: 'Simple Form',
    layout: 'anchor',
    defaultType: 'textfield',
    items: [{
        xtype: 'panel',
        title: '75% Width and 20% Height',
        anchor: '75% 20%'
    }, {
        xtype: 'panel',
        title: 'Offset -300 Width & -200 Height',
        anchor: '-300 -200'     
    }, {
        xtype: 'panel',
        title: 'Mixed Offset and Percent',
        anchor: '-250 20%'
    }],
    bodyPadding: 5,
    width: 350,
    renderTo: Ext.getBody()
})
