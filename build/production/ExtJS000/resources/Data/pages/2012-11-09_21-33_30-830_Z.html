Ext.layout.container.Column

Another common layout type used when creating forms is the Column layout. Column layouts enable you to place fields into two or more columns.

Ext.create('Ext.form.Panel', {
    title: 'Column Layout',
    bodyPadding: 8,
    labelWidth: 80,
    width: 550,
    layout:'column',
    defaults:{
        xtype: 'textfield',
        padding: 3,
        columnWidth: 0.4,   //change this to 0.33      
    },
    items: [{
        fieldLabel: 'First Name',     
    },{
        fieldLabel: 'Middle Name',
    },{
        fieldLabel: 'Last Name',

    },{
        fieldLabel: 'Nick Name',
    }],
    renderTo: Ext.getBody()
});
