You can also create an instances of a class via an alias. This is the preferred alternative to using Ext.create().

All components have a special alias called an xtype. Other classes such as layouts also have aliases.

In this example, two aliases are used: button and vbox.

Ext.create('Ext.panel.Panel', {

	items : [ 
        Ext.create('Ext.button.Button', {
           text: 'Button'
        })
     ],

	layout: 'vbox',
	title: 'My Panel',
	height: 300,
	width: 300,
	bodyPadding: 8,
	renderTo : Ext.getBody()
});
 
Ext.create('Ext.panel.Panel', {

	items : [ {
		xtype: 'button',
		text: 'Button'
	}],

	layout: 'vbox',
	title: 'My Panel',
	height: 300,
	width: 300,
	bodyPadding: 8,
	renderTo : Ext.getBody()
});
