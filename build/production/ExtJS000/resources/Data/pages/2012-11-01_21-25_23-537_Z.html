We can provide additional data to this template through the use of renderData, by default renderData will already contain several properties: $comp, id, ui, uiCls, baseCls, componentCls & frame. Though we can add any additional properties we wish access to within the template.

Ext.create('Ext.Component',{
    renderTpl: '<ul><li>{firstItem}</li><li>{secondItem}</li></ul>',
    renderData: {
        firstItem: 'This one goes first',
        secondItem: 'and this one second'
    },
    renderTo: Ext.getBody()
});
