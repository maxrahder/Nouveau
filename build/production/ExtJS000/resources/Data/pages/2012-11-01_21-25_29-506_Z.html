The final property to review when utilizing renderTpl is childEls. This allows us to define elements within the generated HTML that we'll need references to later on. The framework expects these elements to have a specific id property that is the component id followed by '-' then the property name we wish to use for reference

for example {id}-first

We can then reference component.first and will have an Ext.Element object we can use to manipulate that html

var comp = Ext.create('Ext.Component',{
    renderTpl: '<ul><li id="{id}-first">{firstItem}</li><li>{secondItem}</li></ul>',
    renderData: {
        firstItem: 'This one goes first',
        secondItem: 'and this one second'
    },
    childEls: ['first'],
    renderTo: Ext.getBody()
});

var i = 0;
Ext.create('Ext.Button',{
    text: 'Update first list item',
    handler: function() {
        comp.first.update('Updated first ' + (++i));
    },
    renderTo: Ext.getBody()
});
