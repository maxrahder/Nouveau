MVC allows you to separate control logic.
Here, the button is selected via Ext.ComponentQuery selector syntax within the controller. This is similar to a CSS selector, but it's designed for selecting components rather than DOM elements. 

Ext.define('MyApp.controller.BeatlesController', {
    extend : 'Ext.app.Controller',
    ...
    init : function() {
        this.control({
            'beatlesgrid #sheLovesMe' : { 
                click : this.yeahYeahYeah 
            }
        });
    },
    yeahYeahYeah : function(button) {
        Ext.Msg.alert(button.getText(), 'Yeah yeah yeah!');
    }

});
