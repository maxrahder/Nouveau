Ext.dom.Element
  • A DOM element wrapper, returned by Ext.fly() and Ext.get()
  • Adds animation, event and styling methods
Ext.dom.CompositeElement
  • A collection of Element objects, returned by Ext.select()
  • Ext.select() uses CSS selector syntax (see Ext.dom.Query for syntax)
  • Has the same methods as Element, but runs the method on all its objects
Ext.dom.DomHelper
  • Used to create DOM elements
