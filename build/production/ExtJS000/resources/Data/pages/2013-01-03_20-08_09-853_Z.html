Component selectors are also available in convenience methods on Ext.Component and Ext.container.Container. Use down() to find the first component under a container, and up() to find the first container above a component.

For example, a typical coding idiom for a form panel save button is:

onSaveClick : function(button){
    button.up('myform').submit();
}

Or combine method calls to go to a parent container, and search down from there. 

onClick : function(button){
    button.up('myform').down('toolbar #someButton').disable();
}
