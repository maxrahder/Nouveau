Sencha Cmd

You will need to have Sencha Cmd 3.1 (or higher) installed.

Sencha Cmd 3.1 removes the need to have Sass and Compass installed on your computer since it uses its own bundled version of Sass and Compass.

Ruby

To build themes using Sass on Windows, download and install Ruby. Ruby is already provided by Mac OS X.

Be sure that you have either ruby 1.8.7 or 1.9.3 with ruby -v

Ruby 2.x will not work!

See appendix: Appendix > Setup and Organization > Install Ruby
