As you know, event handling is done via the Ext.util.Observable mixin.

In a MVC application, you typically add the event handler using a component selector in the controller's init() method. But if you have a reference to an observable object, there is other syntax for adding an event handler.

You add listeners via:

  • The listeners config
  • The on() or addListener() method

You remove listeners via:

  • The un() or removeListeners() method
  • The clearListeners() method
