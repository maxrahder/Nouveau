For theming an Ext JS 4.2 application in combination with Sencha Cmd 3.1.*, you will use Sass and Compass. However since the release of Sencha Cmd 3.1.* it's not required anymore to have Sass and Compass installed on your system. Generating Sass files will be handled by Sencha Cmd. You do need to have a working version of Ruby installed on your machine. To confirm that Ruby is running enter in your console the following commands:

ruby -v

Be sure that you have either ruby 1.8.7 or 1.9.3

Ruby 2.x will not work!

If everything is correctly installed you will be prompted with the latest version. If not, follow the instructions below.

Mac OSX

Ruby is already installed on Mac OSX

Windows

  • Open a browser to http://rubyinstaller.org
  • Click the big red Download button
  • Click the link for Ruby 1.9.* (don't choose Ruby 2.x!)
  • Run the installer
  • Accept the default license
  • Turn on the checkbox to add Ruby to your path
  • Turn on the checkbox to associate .rb and .rbw files
  • Click Install

Add Ruby to your paths

If using OSX, you can add the Ruby path by typing the following at a command prompt:
Note that etc directory is hidden.

sudo vi /etc/paths

You will then add the path to the file using the vi editor. Typically the path resembles the following:

/Users/username/.gem/ruby/1.8/bin

The VI Editor has the following commands:

Command Description
*iInsert text before cursor, until is pressed
:wq Save the modified file and quit vi
*u Undo
  • If you are using windows, you can run the installer. You should check the checkbox to make a class path to Ruby.
  • You will need to restart Windows after completing the Ruby install process.
