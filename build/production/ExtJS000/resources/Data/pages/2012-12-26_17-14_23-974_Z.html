Region weights determine the priority of determining the order in which width and height are determined. The region with the greater weight is allocated its height or width first. Weight is specified via regionWeights on the parent container, or — more commonly — via weight on a child. 

Ext.create('Ext.panel.Panel', {

    layout : 'border',

    items : [ {
        title : 'West',
        region : 'west',
        weight : 20,
        width : 100
      }, {
        title : 'North',
        region : 'north',
        weight : 10,
        height : 60
    }, {
        title : 'East',
        region : 'east', 
        weight : 5,
        width : 60
    }, {
        title : 'South',
        region : 'south',
        weight : -20,
        height : 60
    }, {
        title : 'Center',
        region : 'center'
    } ],

    title : 'Border Layout Panel',
    bodyPadding : 8,
    height : 340, width : 400,
    renderTo : Ext.getBody()
});
