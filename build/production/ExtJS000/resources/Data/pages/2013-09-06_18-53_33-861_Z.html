Introduction

In this lab you will have your application use the Neptune theme.

Objectives

  • Change the application theme

Steps

  1. Review how the application references its style sheet

    Open YelpExtplorer/index.html and note that it references bootstrap.css.

    Open YelpExtplorer/bootstrap.css and note that by default, it references the classic theme package in your workspace's ext/packages folder.

  2. Change your application's theme to Neptune

    Open app.json and add this property:

    "theme": "ext-theme-neptune"

    In your terminal window, navigate to your application root folder and build your application via: 

    sencha app build testing

    This will create a new, non-minimized CSS file based on the specified theme, along with any application-specific theming. We don't have any application-specific theming yet — we'll add that in the next lab.

    Open YelpExtplorer/bootstrap.css again and note that it now points to the newly generated theme.

  3. Refresh your application

    In your web browser visit or refresh YelpExtplorer. You should see the Neptune theme.
