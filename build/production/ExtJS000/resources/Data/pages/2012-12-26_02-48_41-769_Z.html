• You can try out the examples that follow in the console of your browser.
  • If you enter 1 + 2 in the console, you should see 3 as the result: 
> 1 + 2
  3
  • JavaScript uses a read-eval-print loop This means that the console
    1. reads in the string "1 + 2"
    2. the reader then converts 1 and 2 to numbers and + to the plus operator,
    3. evaluates the result from the expression, and
    4. prints the returned results out back to the console.
  • Note that when you enter var x = 4 in the console, the next line says undefined
  • In this case, var statement returns nothing, so JavaScript prints undefined out to the console. 
> var x = 4
  undefined
