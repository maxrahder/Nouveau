[workspace root]/packages/my-theme/

The build of the theme.
This directory contains any JavaScript overrides to Ext JS Component classes that may be required for theming those Components.
This directory contains images and other static resources.
  • etc/ contains additional utility functions or mixins
    Only sass/etc/all.scss is processed, although it may call other etc mixins
  • src/ contains Sass rules and UI mixins
  • var/ contains Sass variables
    Global theming typically goes in var/Component.scss
This is the package properties file. It tells Sencha Cmd certain things about the package like its name, version, and other packages that it requires.
