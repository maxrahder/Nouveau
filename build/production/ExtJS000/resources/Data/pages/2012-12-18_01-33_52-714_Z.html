You can create models automatically with Sencha Cmd version 3 and higher, as long as the app was originally created with sencha generate app. Enter the following in a command prompt, in the app's top level directory where "index.html" sits. (Note: The "fields" value cannot contain spaces. In this example, we are also assigning a data type to the lastName field created.) 

sencha generate model Person firstName,lastName:string

The sencha generate command generates app/model/Person.js with this content:

Ext.define('MyDemoApp.model.Person', {
    extend: 'Ext.data.Model',
    fields: [
        { name: 'firstName', type: 'auto' },
        { name: 'lastName', type: 'string' }
    ]
});
