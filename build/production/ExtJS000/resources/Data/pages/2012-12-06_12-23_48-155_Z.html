There are two main types of Proxy:

  • Clientside Proxies
    • MemoryProxy

      Saves data in memory Data will be gone after refreshing the page.

    • SessionStorageProxy

      Saves (if browser supports it) data in Session. Data will be saved only in the browser session.

    • LocalStorageProxy

      Saves (if browser supports it) data in LocalStorage. Data will be saved for a defined period.

  • Remote Proxies
    • Ajax

      sends requests to a server on the same domain

    • JsonP

      uses JSON-P to send requests to a server on a different domain

    • Rest

      uses RESTful HTTP methods (GET/PUT/POST/DELETE) to communicate with server

    • Direct

      uses Ext.direct.Manager to send requests
