You can only use sencha app build on an application originally created using Cmd. To convert an application you hand-coded into an application you can build:

  • Use Cmd to create a new workspace
  • Use Cmd to create a new application with the same name as your application
  • Copy your application's resources to the new application
  • Copy your application's app/ folder to the new application (and remove the generated code)
  • Recreate the application's bootstrap files via sencha app refresh
  • That's it!

This procedure only works if your application is property structured according to Sencha's Compiler-Friendly Code Guidelines.
