The panel header itself is simply a docked item: When you specify title, the panel creates a Ext.panel.Header component, puts the title into the header, then docks it to the top of the panel. Other docked items are added after that.

So this panel actually has three docked items: the header (top), the toolbar (top) and the button (bottom). 

Ext.create('Ext.panel.Panel', {

    title : 'My Panel', // Placed in the Ext.panel.Header, docked at the top
    headerPosition : 'top', // Defaults to 'top' -- specifies the Ext.panel.Header dock position
    height: 200, //height is required to position the header left. 
    dockedItems : [ {
        xtype : 'toolbar',
        dock : 'top',
        items : [ {
            text : 'Toolbar Button'
        }]
    }, {
        xtype : 'button',
        dock : 'bottom',
        text : 'Bottom Button'
    } ],
    
    width : 300,
    bodyPadding : 16,
    layout : 'vbox',
    items : [ {
        text : 'Center Button',
        xtype : 'button'
    }],

    renderTo : Ext.getBody()
});
