• Each function gets a special local variable called arguments.
  • This contains an array-like object containing each of the arguments passed into the function call.
  • This object does have a length property.
> function abc() {
    for ( var index = 0; index < arguments.length; index++ ) {
          console.log( arguments[index] )
    }
    return arguments;
 }
  undefined
> abc( 'A', 'B' ) // outputs A B and returns ["A", "B"]
  A
  B
  ["A", "B"]
> abc() // just returns an empty array-like object
  []
  • The callee property of the arguments object returns the function object itself.
> function xyz() { return arguments.callee.name; }
  undefined
> xyz() // outputs "xyz"
  "xyz"
