Here's a container (a form panel), containing several components (input fields and buttons). The Save button fires a click event. 

Ext.create('Ext.form.Panel', {
	title : 'Simple Form',
	bodyPadding : 6,
	width : 300,
	height: 200,
	defaultType : 'textfield',
	layout : 'anchor',
	items : [ {
		fieldLabel : 'First Name',
		name : 'first'
	}, {
		fieldLabel : 'Last Name',
		name : 'last',
		allowBlank : false
	}, {
		xtype : 'datefield',
		fieldLabel : 'Date of Birth',
		name : 'dob'
	}, {
		xtype : 'checkboxfield',
		boxLabel : 'Married',
		name : 'married'
	}],
	buttons : [
		{ text : 'Save', handler : function(){
                alert('Saving');
             }
        }, 
        { text : 'Cancel'}
	],
	renderTo : Ext.getBody()
});
