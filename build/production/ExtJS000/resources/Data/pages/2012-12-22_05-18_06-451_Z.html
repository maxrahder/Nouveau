• Some Ext JS methods have optional scope parameters.
  • As in the case of Ext.bind(), setting the scope parameter changes the value of this inside the function from the normal value of this to the value of the scope parameter.
> var aFn = Ext.bind( function(){return this + 1}, 2 );
  undefined
  // 2 is the scope parameter. It becomes the value of 'this'.
> aFn() // returns 2 + 1
  3
