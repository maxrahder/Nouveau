Ext.String.ellipsis(value,length,[canBreakAfterWord]):String

Truncate a string and add an ellipsis ('...') to the end if it exceeds the specified length.

var a = 'The rain in Spain falls mainly on the plain.';
alert(Ext.String.ellipsis(a, 35));
alert(Ext.String.ellipsis(a, 35, true));

Note that this truncates based on character count, regardless of the typographic length of the string.
