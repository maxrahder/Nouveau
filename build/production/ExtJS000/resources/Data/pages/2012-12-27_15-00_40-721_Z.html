• Layouts arrange items within a container
  • You specify a layout via the layout property on the container; each container has a default layout
  • Layouts are only specified via their alias — you never use Ext.create() to create a layout
  • Using the alias name alone creates the layout using default settings; passing an object creates the layout with the specified config
  • Some layouts require additional information tagged on each item, such as the region used with border layout
