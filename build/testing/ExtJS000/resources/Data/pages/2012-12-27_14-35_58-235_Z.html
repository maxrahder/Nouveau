Anchor layout arranges items top to bottom, where each takes up a relative width and height, specified via the anchor value on each item. The size is relative to the container — resize the window to see this in action.

The value specifies a width and (optionally) height. The values can be either an integer offset or a percentage. For example, an item using anchor: '-50 75%' will take up 50 pixels less width than the container width, and 75% less height than the container height. 

Ext.create('Ext.window.Window', {

    layout : 'anchor',
    items : [ {
        style : {'background-color' : 'red'},
        anchor : '90% 20%'
    }, {
        style : {'background-color' : 'white'},
        height : 10,
        anchor : '-10'
    } , {
        style : {'background-color' : 'blue'},
        anchor : '90% 20%'
    } , {
        style : {'background-color' : 'green'},
        anchor : '-50 40%'
    } ],

    title : 'Window',
    defaults : {xtype : 'component'},
    height : 300,
    width : 300,
    bodyPadding: 6,
    x : 1, 
    y : 1,
    autoShow : true

});
