The base class for all user-interface widgets. Components have height, width, simple styling via padding, margin and style, and some CSS styling properties.

The base class for all containers. Containers contain components, specified via the items array. Items are arranged via the container's layout manager.

Containers are components, so have have all component styling, and bodyPadding and bodyMargin as well.

Provides event handling to components, data stores and some other classes. Events are fired via the fireEvent() method. Listeners are added via the on() or addListener() method, or via configuration settings.
