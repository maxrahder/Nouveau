1. Download and Install Eclipse

    2. Download Eclipse Indigo IDE for Java EE Developers

  2. Unzip the Eclipse zip file to /Applications.

    This creates a subdirectory named eclipse that contains multiple subdirectories.

  3. Download the Sencha Complete Trial

  4. Run the Sencha Complete trial. Make a note of the installation folder.

  5. Open the directory where you installed Sencha Complete. Unzip the Sencha Eclipse plugin to the directory of your choice.

  6. Start Eclipse.

  7. Choose Help > Install New Software

  8. In the "Work with" section, click the Add... button.

    9. The Add Repository dialog box appears.

  9. Click Archive and select the Sencha_Eclipse_Plugin-trial-version.zip directory where you unzipped the Sencha Eclipse plugin.

  10. Enter the name: Sencha Plugin

  11. Click OK.

    You should see a reference to Sencha Eclipse Plugin- 30 Day Trial appear in the window.

  12. Check the checkbox adjacent to Sencha Eclipse Plugin-30 Day Trial.

  13. Click Next.

  14. Click Next.

  15. Accept the terms of the software license.

  16. Click Finish.

    VJET and the Sencha Eclipse plugin will install.

  17. Press OK at the Security Warning prompt.

  18. Restart Eclipse.

    19. Note: When Eclipse restarts you will be greeted with a Welcome Screen which can take you to helpful information on the plugin. Should you close it, it can be reopened under the Help menu.
