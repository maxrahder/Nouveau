These variables are available in a template expression

values The values in the current scope.
parent The scope (values) of the ancestor template.
xindex In a looping template, the loop index (1-based).
xcount In a "for" looping template, the total length of the array you are looping.
xkeyIn a "foreach" looping template, the key of the current property being examined.

For example:

'<tpl for="family">',
    '<tr class="{[( ( xindex % 2 ) === 0 )? "even" : "odd"]}">',
...
'</tpl>'
