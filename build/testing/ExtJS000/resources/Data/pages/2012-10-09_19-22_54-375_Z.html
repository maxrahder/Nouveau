There are also shortcut string values for each. Try 'em!

See API docs
  • Use a literal string for tbtext
  • Use an empty string literal for tbspacer
  • Use a literal dash for tbseparator
  • Use a literal dash greater-than for tbfill
Ext.create('Ext.panel.Panel', {
    title : 'My Panel',
        
    tbar : [ 'The Beatles', 
        '', {
        xtype: 'button',
        text : 'John'
    }, {
        xtype : 'tbspacer',
        width: 32
    }, {
        xtype: 'button',
        text : 'Paul'
    }, '-', {
        xtype: 'button', 
        text : 'George'
    } , '->', {
        xtype: 'button',
        text : 'Ringo'	
    }],

    height: 200,
    width: 400,
    renderTo : Ext.getBody()

});
