Ext.form.field.Text

Supports empty-field placeholder values (see emptyText). Normally this value will be submitted to the server if this field is enabled; to prevent this you can set the submitEmptyText option of Ext.form.Basic.submit to false.

Ext.create('Ext.form.Panel', {
    title: 'Contact Info',
    width: 300,
    bodyPadding: 10,
    renderTo: Ext.getBody(),
    items: [{
        xtype: 'textfield',
        name: 'name',
        fieldLabel: 'Name',
        emptyText: 'Full Name'
    }, {
        xtype: 'textfield',
        name: 'email',
        fieldLabel: 'Email Address',
        emptyText: 'e.g. name@domain.com'
    }]
});
