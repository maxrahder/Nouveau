Components have a few properties for tagging the component with CSS styles: cls, baseCls, componentCls, disabledCls, overCls, styleHtmlCls

Of these, the cls property is the most commonly used, but none of them are needed very often. 

Ext.create('Ext.Component', {
    html: 'Hello world!',
    // style : {border: 'thin solid red'},
    cls : 'coolborder',
    width: 300,
    height: 100,
    padding: 8,
    margin: 40,
    renderTo: Ext.getBody()
});
