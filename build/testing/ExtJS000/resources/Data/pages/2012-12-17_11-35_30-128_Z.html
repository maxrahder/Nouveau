Introduction

In this lab, you will optimize the css to reduce the size of the YelpExtplorer application.

Objectives

  • Optimize an application's theme to reduce the css file size

Optimize CSS

  1. Build the theme, from the YelpExtplorer folder. 
    sencha theme build YelpTheme
  2. Notice the file size of YelpTheme/build/resources/YelpTheme-debug.css.

    This file is automatically generated by Sencha Cmd, with the output_style set to :expanded.

  3. Notice the file size of YelpTheme/build/resources/YelpTheme-all.css.

    This file is automatically generated by Sencha Cmd, with the output_style set to :compressed.

  4. Open /build/YelpTheme/production/YelpExtplorer-all.scss in your editor.

    Notice that all the components that are used in your application are set to true. This will reduce your stylesheet's size.
