Use the support portal to create a support ticket when you need assistance. Please provide Sencha the following information, so we can easily help you out.

  • Sencha product component name and issue description in subject line of ticket

    For example: Ext JS Grid displays only one record.

  • Sencha product and product version you are using.

    For example: Ext JS 4.2.0

  • Browser and browser version in use when you noted an issue

    For example: IE7

  • Test case that we can use to reproduce the issue
    • A simple example that shows an issue in a Sencha product
    • Reproducible
    • In a supported environment
    • Isolated from third party products
    • Without requirements to access any remote resources
  • Log files or error reports
  • Any screen shots or short video clips that clearly show the behavior in questions
