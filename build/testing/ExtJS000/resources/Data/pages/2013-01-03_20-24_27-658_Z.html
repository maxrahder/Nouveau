To do this, have the view provide event handler for the user actions, then have the parent throw the logical event. The fireEvent() method is part of the Ext.util.Observable mixin — all components are observable. 

...
{
    text : 'Search',
    icon : 'app/resources/icons/find.png',
    itemId : 'twitterSearch',
    handler : function(b){
        b.up('twittergrid').fireSearch();
    }
}
...
fireSearch : function(){
    var s = this.down('#searchText').getValue();
    this.fireEvent('search', this, s);
}
