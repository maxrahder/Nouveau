• Brendan Eich was recruited by Netscape in 1995 to quickly create a scripting language for their browser. Netscape decided that it should look something like the fast rising Java language. Originally named Mocha, then LiveScript, it was finally named JavaScript. ( http://en.wikipedia.org/wiki/JavaScript )
  • JavaScript was heavily influenced by Scheme, a dialect of LISP (with first-class functions), and Self (with prototypes). ( http://en.wikipedia.org/wiki/JavaScript )
