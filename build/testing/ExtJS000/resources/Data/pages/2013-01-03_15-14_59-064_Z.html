Let's take a closer look at the requires array. The requires array isn't part of MVC per se, but understanding it helps understand how the pieces of an application are in memory when you need them.

Ext.define('MyApp.view.Viewport', {
    extend : 'Ext.container.Viewport',    
    requires : [ 'MyApp.view.BeatlesGrid' ],
    layout : 'fit',
    items : [ {
        xtype : 'beatlesgrid',
        store : 'People'
    } ]
});
