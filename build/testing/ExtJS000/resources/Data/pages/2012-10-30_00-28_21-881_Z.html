Ext.Array.merge(array1, array2, etc)

Merge multiple arrays into one with unique items.

union() is an alias for merge().

var a = [ 1, 2, 3 ];
var b = [ 2, 3, 4 ];
var c = [ 2, 3, 5 ];
var d = [ 2, 3, 6 ];
var e = Ext.Array.merge(a, b, c, d);
alert(e); // alert [1, 2, 3, 4, 5, 6] - all items in a new array
