As you've seen, Ext JS dynamically loads files as they are referenced — if a file is not already in memory, it's loaded.

This applies to all classes being references, both your own classes and classes in the Ext JS library. In other words, if you create a Ext.panel.Panel, and it's not already in memory, its source file will be dynamically loaded.

Question: But won't the library classes already be in memory?

Answer: It depends on what library you're using.
