Tab panels are special panels that use card layout. Typically, the items are panels. For each child item, the tab panel the item's title for the tab caption. The key configs are:

  • activeTab (defaults to 0)
  • deferredRender
  • removePanelHeader
  • tabPosition
Ext.create('Ext.tab.Panel', {
    renderTo : Ext.getBody(),
    height : 200,
    width : 200,
    //tabPosition: 'bottom',
    //removePanelHeader: false,
    plain : true,
    items : [{
        title : 'One',
        html : 'Panel One'
    }, {
        title : 'Two',
        html : 'Panel Two'
    }]
});
