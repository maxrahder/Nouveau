• In JavaScript, functions are not just values, they are also JavaScript objects. This means that they have properties.
  • If you just enter the name of the function, you see the code.
> function abc() { return 123; }
> abc // returns "function abc () { return 123; }"
  • But, in some browsers (Chrome, Safari, Firefox, Opera), the function object has a name property.
> abc.name // returns "abc"
