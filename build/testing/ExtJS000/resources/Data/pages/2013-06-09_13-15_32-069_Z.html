The proxy's api config allows you to send unique URLs to the back end, so it can receive and differentiate between records being created, updated and destroyed.

The api config may be used along with url — the url is the default for any missing api value.

Ext.define('MyApp.model.Person', {
    extend : 'Ext.data.Model',
    proxy : {
        type : 'ajax',
        api : {
            read :    'backend.php?mybackendparameter=retrieve',
            create :  'backend.php?mybackendparameter=create',
            update :  'backend.php?mybackendparameter=update',
            destroy : 'backend.php?mybackendparameter=delete'
        }
    },
    fields : [ 'name' ]
});
