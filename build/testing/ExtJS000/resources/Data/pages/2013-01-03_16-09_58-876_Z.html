The point is, the Ext.app.Application controllers array and the Ext.app.Controller models, stores and views arrays, all essentially do a "requires" call. 

Ext.define('MyApp.controller.BeatlesController', {
    extend : 'Ext.app.Controller',
    views : [ 'BeatlesGrid' ],
...

Is equivalent to 

Ext.define('MyApp.controller.BeatlesController', {
    extend : 'Ext.app.Controller',
    requires : [ 'MyApp.view.BeatlesGrid' ],
...
