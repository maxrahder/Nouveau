The "index.html" file contains three things:

  • A reference to a Ext JS style sheet
  • A reference to the Ext JS library
  • A reference to "app.js" 
    <head>
  <link rel="stylesheet" type="text/css" href="/packages/ext-theme-neptune/build/resources/ext-theme-neptune-all.css">
  <script src="/ext/ext-dev.js"></script>
  <script src="app/app.js"></script>
</head>
