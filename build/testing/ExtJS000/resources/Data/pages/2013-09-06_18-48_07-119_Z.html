Running sencha app build creates a new .css file that reflects the application's specified theme package and any additional theming done in the application's sass/ folder.

By default, the generated .css is minimized. Specify testing to generate an un-minimized .css file.

Only CSS classes used by your application are included. For example, if your application doesn't have a tree panel, then no tree panel CSS classes are included.

(Cmd also creates a .js file of those classes used by your application — we'll talk about that in the next topic.)
