The xtype configuration property is used for components. For example, The xtype for Ext.button.Button is 'button'.

Using xtype is the most common way to define component instances, especially in a container.

Ext.create('Ext.panel.Panel', {

	items : [ {
		xtype: 'button', 
		text: 'Button'
	}],

	layout: 'vbox',
	title: 'My Panel',
	height: 300,
	width: 300,
	bodyPadding: 8,
	renderTo : Ext.getBody()
});
