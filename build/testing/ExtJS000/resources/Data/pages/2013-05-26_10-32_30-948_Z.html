This course will make use of the Nouveau Training portal, where the student will have access to. It contains presentation slides which will be used during the lectures and details slides which can be used as reference guides. (See the Appendix).

  5. Data aware components
  6. Theming
  7. Going into production

Sometimes topics need more in depth information. We will use the Advanced topic, to refer to some detailed information or examples.
