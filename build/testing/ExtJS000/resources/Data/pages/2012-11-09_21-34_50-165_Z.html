Ext.form.FieldContainer

A field container is a simple container that includes the Ext.form.Labelable mixin. 

Ext.create('Ext.form.Panel', {
    title: 'Ext.form.FieldContainer',
    items: [ {
        xtype: 'textfield',
        fieldLabel: 'Number'
    }, {
        xtype: 'textfield',
        fieldLabel: 'Street'
    }, {
        xtype: 'fieldcontainer',
        fieldLabel: 'City, State ZIP',
        layout: 'hbox',
        items: [
            {xtype: 'textfield'}, 
            {xtype: 'tbtext', text: ', '}, 
            {xtype: 'textfield', width: 30}, 
            {xtype: 'tbtext', text: ' '}, 
            {xtype: 'textfield', flex: 1}
        ]
    }],
    defaults: {anchor: '100%'},
    bodyPadding: 8,
    width: 400,
    renderTo: Ext.getBody()
});
