• Toolbars are usually docked in panels:
    • Via the panel's dockedItems array
    • Or, more commonly, via tbar, bbar, rbar or lbar
  • Toolbars are containers, so in theory, can contain any component
  • Toolbars typically contain buttons
  • There are also special components designed for toolbars:
    • tbText, tbSpacer, tbSeparator, tbFill,
    • Or use literals: 'My Text', '', '-', '->'
