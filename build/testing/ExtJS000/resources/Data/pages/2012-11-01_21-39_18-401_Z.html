Ext.Number.constrain(number, min, max):Number

Checks whether or not the passed number is within a desired range. If the number is already within the range it is returned, otherwise the min or max value is returned depending on which side of the range is exceeded.

alert(Ext.Number.constrain(11, -20, 20)); // alerts 11
alert(Ext.Number.constrain(21, -20, 20)); // alerts 20
alert(Ext.Number.constrain(-21, -20, 20)); // alerts -20
