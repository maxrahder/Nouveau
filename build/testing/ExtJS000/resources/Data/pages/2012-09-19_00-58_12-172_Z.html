• Ext JS uses these powerful low-level classes to create panels, buttons and the other components you need in your application
  • You may occasionally use them
    • If you code direct Ext.Component subclasses you'll probably needs to get into the details of the DOM
    • If you have legacy content, you can use these commands to select and manipulate that content
    • But the whole point of a component library is so you don't need to worry about the DOM very much!
