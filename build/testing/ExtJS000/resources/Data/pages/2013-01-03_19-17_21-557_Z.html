Controllers have a refs property that allows you to put a name on a selector. The name can then be used as a getter in your controller methods.

For example, if you had a text field with the itemId mySearchField, you could code this ref: 

refs : [ {
    ref : 'searchField',
    selector : 'mypanel #mySearchField'
} ]

As a result, in your controller methods you can then get a reference to the component via this.getSearchField().
