• We do not intend this unit to be an introductory course in Web programming. You should have already had that before taking this course. Rather, is intended to highlight some of what Web programmers need to know to write code in ExtJS.
  • Some of what we will cover includes:
    • functions as objects
    • function arguments
    • special variables such as arguments and this
    • scope
    • this.callParent(arguments) which is Sencha's super() method
    • Sencha objects
    • prototypal chaining
