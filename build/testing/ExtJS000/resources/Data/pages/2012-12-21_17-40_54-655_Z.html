Containers arrange their components using a layout manager.

The various layout classes are found in the Ext.layout.container package. 

Ext.create('Ext.panel.Panel', {

    layout : 'vbox',
    //layout : 'hbox',
    items : [ {
        text : 'Button One'
    }, {
        text : 'Button One'
    } ],

    title : 'My Panel',
    defaults : {
        margin : 4, xtype: 'button'
    },
    bodyPadding : 8,
    renderTo : Ext.getBody()
});
