There are several good JavaScript debugging environments available for web browsers. The two most popular are:

  • Firefox uses the Gecko layout engine
  • For debugging, use the Firebug plugin
  • There is also the Illuminations for Developers plugin, which is an ExtJS-aware debugger.
  • Chrome uses the WebKit layout engine
  • Safari and the Chromium open-source browser also use WebKit
  • Opera also plans on moving to WebKit
  • WebKit browsers share similar debuggers
  • Google's tools are documented at the Google Developer Tools
  • There is also the Illuminations for Developers plugin, which is an ExtJS-aware debugger.
