• You can give functions new properties, just like you can give any object a new property.
> function abc() { return 123; }
  undefined
> var abc.bigName = "ABC";
  undefined
> abc.bigName // returns "ABC"
  "ABC"
  • The dev version of ExtJS adds a displayName property to function objects for methods in components. Use Ext.getDisplayName(function) to access it.
> var b = Ext.create( 'Ext.Button' );
  undefined
> Ext.getDisplayName( b.addListener )
  "Ext.AbstractComponent#addListener"
  • In the case above, there is an addListener method defined in Ext.AbstractComponent. So Ext.getDisplayName(b.addListener) returns the displayName of the addListener method/function/object found in prototype property of the AbstractComponent class. (see later, Prototypes)
