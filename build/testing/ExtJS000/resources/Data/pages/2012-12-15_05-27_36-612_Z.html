• The class for a tree is Ext.tree.Panel.

  • You may add an object to the tree panel's root property.

  • The value of the object's text property appears as node's label in the tree.

Ext.create('Ext.tree.Panel', {
    root: {
        text: 'Root Node',
        leaf: true
    },
    title: 'Tree With Root',
    renderTo: Ext.getBody()
});
