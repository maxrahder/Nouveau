Introduction

In this walkthrough, you will work with Sencha Architect 2 for Ext JS projects.

Together with your instructor you will perform the following tasks:

  • Working with column templates

Steps

Add new fields to the model

  1. In the Business model, click [+] next to Fields

  2. Type the following fields, separated by commas:

    • rating_img_url_small

    • photo_url

  3. Drag both columns above MyAjaxProxy.

Convert the Rating column to a template column

  1. In Project Inspector, select the GridPanel and right-click the Rating column.

  2. Hover over Transform

  3. In the sub-menu, choose Ext.grid.column.Template

Define the template

  1. Click on Rating column

  2. Click the [+] next to tpl property, then click the → (arrow icon).

    This will place you in the editor where you can edit the template. In the editor, type:

    <img src="{rating_img_url_small}">

  3. Return to Design view, and verify that the ratings graphics appear.

    Note: Should the content not display in Design view, Preview the application in your browser. If the grid is blank, you may need to right-click on the Businesses store and choose Load Data.

Save and Preview the Application

You should see rating stars in the Rating column and a formatted phone number in the Phone column.

Conclusion

Templates are a very powerful feature. A small amount of html code will parse all data supplied to it and render all items.
