The base class for models is Ext.data.Model. To define your own models, use the following code: 

Ext.define('MyApp.model.MyModel', {
    extend : 'Ext.data.Model',

    fields : [ ... ],

    validations: [ ... ], // optional

    customFunc: function() {
		... custom function body ...
    },

    proxy: {
        type: 'ajax',
        url: 'http://localhost'
    }
});
