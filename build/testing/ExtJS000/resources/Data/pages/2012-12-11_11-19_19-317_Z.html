Introduction

In this walkthrough, you will continue to work with Sencha Architect to create a working data grid with the data that you connected to previously.

Together with your instructor, you will perform the following tasks:

  • Define a Viewport

  • Define a Grid

  • Bind a Store to a Grid

  • Setup Grid columns

This lab requires the Sencha Architect project files created in the previous Walkthrough 1. Lab: Data Connection of this topic.

Steps

Define a Viewport

  1. From the toolbox, drag a Viewport into the Project Inspector, (Views section).

  2. In the Project Inspector, click on MyViewport you just added

Define a Grid

  1. In the Toolbox, type gr into the filter field.

  2. Double click on Grid Panel to add it as a child under the Viewport.

  3. Go to Design view.

  4. Double-click on the Grid Panel header and give it the title: Businesses

  5. Configure the following properties:

    • store: Businesses (select from the drop down)

    • margin: 20

    • height: 300

    • autoShow: true (Check the box)

Load Data into the Grid

  1. Right-click on the Businesses grid, and choose the Auto-columns option on the menu.

    You should see the grid populated with the columns from the model.

    Note: If the data is not visible, right-click on the Businesses store and choose Load Data.

  2. Remove the url column by right-clicking on url in the Project Inspector and selecting delete.

    Note: To rename a column, click on the column in the Project Inspector and change the columnâ€™s text property. You can also double-click on the column's header.

  3. Double-click on the Full_address header in Design view and change the header to Address. You can tab to the remaining fields that need to be changed

    • Change Avg_rating to Rating

    • Change Review_count to Reviews

  4. Select Name and Address columns and set the flex property in the Config panel to flex: 1

  5. Set the align property for the Rating column to center

    Note: To make the next step easier, type ali in the filter field for the Config panel, and click on the lock icon. This will apply the filter automatically as you change between columns.

  6. Set the align property for the Reviews column to right.

    Note: As a shortcut, you can apply focus to the drop down and type l, c or r (for left, center and right alignment)

  7. Set the format property for the Reviews column to 0.

  8. In the Project Inspector, drag the Reviews column and drop between the Address and Rating columns.

Save and Preview the Application

Note: You can sort the data by clicking on the column headers. You can hide columns. Click on the arrow in the column header. Hover over Columns in the drop down menu. Uncheck the column you want to hide.

Conclusion

You have just created a working grid! It's not cheating to work smarter than harder. Congratulations!
