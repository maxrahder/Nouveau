Ext.isEmpty(value, [allowEmptyString]):Boolean

Returns true if the passed is empty, false otherwise.

out(
Ext.isEmpty(),          // true
Ext.isEmpty(null),      // true
Ext.isEmpty(undefined), // true
Ext.isEmpty([]),        // true
Ext.isEmpty(''),        // true
Ext.isEmpty(false),     // false
Ext.isEmpty(0)          // false
);
