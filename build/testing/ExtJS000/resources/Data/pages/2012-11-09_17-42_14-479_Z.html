Ext.form.field.TextArea

Grows in height to fit its content. (Useful configuration options - growMin and growMax. Allow you to set the minimum and maximum grow heights.) 

Ext.create('Ext.form.FormPanel', {
    title      : 'Sample TextArea',
    width      : 400,
    bodyPadding: 10,
    renderTo   : Ext.getBody(),
    items: [{
        xtype     : 'textareafield',
        grow      : true,
        name      : 'message',
        fieldLabel: 'Message',
        anchor    : '100%'
    }]
});
