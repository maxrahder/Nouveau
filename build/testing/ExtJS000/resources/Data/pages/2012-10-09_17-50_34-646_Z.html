The collapsible config specifies if a panel is collapsible. Setting titleCollapse to true lets the user click anywhere on the title to collapse or expand the panel. 

Ext.create('Ext.panel.Panel', {
    title : 'My Panel',
    renderTo : Ext.getBody(),
    height : 200,
    width : 200,
    collapsible : true,
    titleCollapse : true
});
