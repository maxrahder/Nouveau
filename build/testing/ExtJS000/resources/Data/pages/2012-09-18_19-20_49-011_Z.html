xtype:panel     Ext.panel.Panel

A panel is a container, and like any container, has items and a layout. 

Ext.create('Ext.panel.Panel', {
	title : 'My Panel',
	renderTo : Ext.getBody(),
	layout : 'vbox',
	items : [ {
		xtype : 'button',
		text : 'Button One'
	}, {
		xtype : 'button',
		text : 'Button One'
	} ]
});
