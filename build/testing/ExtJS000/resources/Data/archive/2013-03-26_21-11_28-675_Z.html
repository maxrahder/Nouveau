We've tried to provide you with some helpful tips that can help to get you up to speed on sublimeText.

Conventions used in these instructions.

The first instruction is for the Mac platform, if there is a PC specific alternative, it will be in parentheses.

Note: The standards are for Cmd to be Ctrl on the PC and Option to be Alt. These will not be listed. Only exceptions will be shown.

Setting up the Workspace

Cmd + B + K will hide the sidebar. May not need it if you get good at locating files with Cmd + P (e.g., can type ignore to find gitignore).

Cmd + Ctrl + Shift + F will take you into distraction free mode - Nothing but editor for full screen. Information on customizing Distraction Free Mode.

Right-side sidebar is a screen shot of all code and lets you navigate quickly to any part of the file, just by clicking on the target code.

Cmd + = Zooms in
Cmd + - Zooms out

For Columns, Cmd + Option + 1, 2, 3 or 4 for columns; Shift + Cmd + Option + 2 or 3 for rows; or Shift + Cmd + Option + 5 for a grid

Ctrl + 1, 2, 3 or 4 allows you to move back and forth between the views. You also have the ability to go into the File Menu and choose New View Into File, then you can drag that into the same new interface so you can have two views of one file but the edits will show in both.
