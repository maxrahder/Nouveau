Your instructor will provide you the course materials: SenchaTraining.zip. Unzip this at the web server document root. (This is the same location where Ext JS was installed.)

Since you’ll be editing the course files, Mac users may need to set permissions to read/write via the command line or the Finder. If you’re using the terminal, navigate to htdocs and type the following command: 

chmod -R a+rx SenchaTraining

Your folder structure in htdocs should look like this:

  • SenchaTraining
    • ExtJS251
      • Labs
      • LabsSolutions
      • ...
