A store is a client-side cache of model objects. The store holds the records that the app will use.

  1. To create a store, click on the [+] button just to the right of the Project Inspector title.

  2. Select a simple Store from the menu. (not a subitem)

  3. Select MyStore in the Project Inspector.

  4. In the Config panel give this new panel the following property values:

    • userClassName: ITunesStore

    • model: ITunesRecord

    • autoLoad: true (checked)

    Note: Your code should look like this:

    Ext.define('ITunesExtplorer.store.ITunesStore', {
    extend: 'Ext.data.Store',

    requires: [
        'ITunesExtplorer.model.ITunesRecord'
    ],

    constructor: function(cfg) {
        var me = this;
        cfg = cfg || {};
        me.callParent([Ext.apply({
            autoLoad: true,
            storeId: 'MyStore',
            model: 'ITunesExtplorer.model.ITunesRecord'
        }, cfg)]);
    }
});
