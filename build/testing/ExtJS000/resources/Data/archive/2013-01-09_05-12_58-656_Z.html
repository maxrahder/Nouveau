A model class defines the fields that make up a model record.

We can define a model by listing the properties in the data.

For an iTunes album, the fields we will use are: name, artist, link, category, price, and image.

  1. To create a new model, click on the [+] button just to the right of the Project Inspector title. Select: Model.

  2. Select MyModel in the Project Inspector.

  3. In the Config panel give this new model the following property values:

    • userClassName: ITunesRecord

    • fields: name, artist, link, category, price, image, id

  4. Select the [Code] button above the Design Canvas/Code View panel (in the upper right corner) to inspect the code Architect has created.
