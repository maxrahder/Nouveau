Templates are designed to be used with objects or arrays. If your data is in a store use Ext.Array.pluck() to extract the values from each record. An array of records is passed to "load" handlers. Or use getRange() to get a range of records. 

myStore.load(function(records) {
    var array = Ext.Array.pluck(records, 'data');
    // Now use the array with the template.
});
