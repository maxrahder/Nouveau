<head>
<link rel="stylesheet" type="text/css" href="/extjs4/resources/css/ext-all.css">
<script src="/extjs4/ext-all.js"></script>
<script>
    Ext.define('MyApp.view.MyWindow', {
        extend : 'Ext.window.Window',
        bodyPadding : 16,
        html : ':-)'
    });
    Ext.onReady(function() {
        Ext.create('MyApp.view.MyWindow', {
            autoShow : true
        });
    });
</script>
</head>

Again, the Ext.define can be in or out of the Ext.onReady, but the code that actually tries to show the window must be run after Ext JS is ready.
