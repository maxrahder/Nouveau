Class Description

Essential Ext JS is a Live Online Training course providing hands-on training and an overview of essential Ext JS concepts. The course is offered online in half-day sessions. Through intensive hands-on labs, you will learn application architecture, how to fetch data from live data feeds, and the use of the key user interface components to build a complete application. You will also build applications with Sencha Architect, Sencha’s visual application builder.

Live Online Training courses are done as half-day sessions given by an Ext JS expert, leading from a remote location. The training will also include daily “office hours” where you can ask questions and get help with the labs.

Target Audience

  • Managers and Developers assessing the ExtJS Framework
  • Developers who want to get an overview of the Framework's capabilities
