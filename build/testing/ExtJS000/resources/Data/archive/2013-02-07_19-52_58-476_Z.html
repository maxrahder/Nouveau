Getting Started

  1. Create a new VJET project by going to File > New > VJET Project...

  2. Create a new project or open one from the existing source.

    3. You may see a dialog for using VJET JS perspective. Agree to open your project in this perspective, if prompted.

    Note: To use Ext JS you need to add a VJET project dependency.

  3. Right click on the new project and select Properties. Select VJET > VJET Build Path . Select Projects tab and click the Add... button to add a project. Select and add ExtJsTL project.
