With your application text now stored as component properties, you can create a script that sets the prompts for a specific language, as illustrated by the following example:

if (MyApp.view.FeedbackForm) {
    Ext.apply(MyApp.view.FeedbackForm.prototype, {
	   lblFirstName: 'pr√©nom',
	   lblLastName: 'nom de famille'
    });
}

Note the following:

  • Always save files containing non-latin characters as UTF-8
  • You may need to allocate additional space for field labels as some translations may require more characters.
  • Right to Left languages (Hebrew, Arabic) may require you to make changes to your stylesheet. In particular, you will likely need to set the css direction property to 'rtl' for the Body tag.
