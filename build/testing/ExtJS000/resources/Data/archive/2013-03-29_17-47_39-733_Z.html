Customization

Preferences

There are two basic areas for preferences information. Default will show you what is available and User is where you should try to make changes that will persist.

Cmd + , gets you to these preference settings.

Settings - Default shows all that you "Can" do but you want to change the User settings instead. You could also go into Settings More - Syntax Specific and then you can have your own settings for various types of file.

Custom Themes
I have added Cobalt, Soda and Refresh. There are many others available on the net. Sublime is also fully compatible with any Textmate themes you may find online.

Key Bindings
Just like with preferences, you can go into the Defaults and see what you want to use and then go to User to assign what you would like to override.

In general, the shortcuts will be listed to the right in the drop down menus. Should you wish to add your own, you can go to Preferences - Keybindings - User and add them.

As an example if you wanted to create a shortcut for Line ReIndent and wanted to use the F12 key you would type in the following:

{"keys": ["f12"], "command": "reindent"}
