You didn't? Here's a work around:
  1. For example if your app is called "MyDemoApp" rename it to "MyDemoApp-backup"
  2. Create a new empty folder: "MyDemoApp"
  3. Generate a new app with Sencha Cmd in the same name space: 
    sencha generate app MyDemoApp /path/to/MyDemoApp
  4. Copy over the sencha app folder
  5. Test it, if it works you can delete the backup
ExtJS will issue warnings to the console for any undeclared dependencies.
.sencha/appsencha.cfg
//added a path to a ux folder.
app.classpath=${app.dir}/app/ux,${app.dir}/app.js,${app.dir}/app
