Ext.Number.snap(value, increment, minValue, maxValue ):Number

Snaps the passed number between stopping points based upon a passed increment value.

The difference between snap() and snapInRange is that snapInRange uses the minValue when calculating snap points.

alert(Ext.Number.snap(56, 2, 55, 65)); // alerts 56
alert(Ext.Number.snapInRange(56, 2, 55, 65)); // alerts 57
