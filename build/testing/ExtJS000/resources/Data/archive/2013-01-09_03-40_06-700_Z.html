1. Open Sencha Architect.

  2. Under the heading Create New Project, select the latest version of Ext JS.

  3. After Sencha Architect opens, Click the Save button on the toolbar.

  4. Set the Save Path to [webserver]/SenchaTraining/ExtJS251/Labs/Architect

  5. Set the Project Name to ITunesExtplorer. (Overwrite if prompted.)

  6. Click Save.

  7. Click Settings button on the main toolbar, and ensure that the Project tab is selected.

  8. Enter the following URL prefix: http://localhost/SenchaTraining/ExtJS251/Labs/Architect/ITunesExtplorer

  9. Browse to the Publish Path: [webserver]/SenchaTraining/ExtJS251/Labs/Architect

  10. Click Save.
