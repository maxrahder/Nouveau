Introduction

During this lab you will start creating your own custom UI's, by using the Ext JS CSS Mixins

  • Use Custom UIs

Note: This lab requires code from the Student files folder: Labs/Theming/LabCustomUI/MyThemeApp/

Steps

  1. First things first. Let's preview the current application in your browser.

    Browse to /Labs/Theming/LabCustomUI/MyThemeApp/. If everything is correct your application should look like the picture below:

  2. Start compass

    Open a command prompt and navigate to LabTheme/Sass/ folder. 

    compass watch .
  3. Open in your editor app.scss from the LabTheme folder.
  4. Create a new mixin to define a custom toolbar called: 'custom-tb' 
    @include extjs-toolbar-ui(
    'custom-tb',
    $background-color: #CEE7F7,
    $background-gradient: null,
    $border-color: null
);
  5. Open app/view/Viewport.js. Where indicated by the comment, add the following configuration attribute to the toolbar: 
    ui: 'custom-tb',
  6. Return to the Sass file and create a new custom mixin to define a new custom panel called: 'custom-bg' 
    @include extjs-panel-ui(
    'custom-bg',
    $ui-body-background-color: #F8F8F8,
    $ui-body-color: #043250,
    $ui-body-font-size: 22px,
    
    $ui-header-background-color: #043250,
    $ui-header-color: #CEE7F7
);
  7. Open app/view/Viewport.js. Where indicated by the comment, add the following configuration attribute to the panel: 
    ui: 'custom-bg',
  8. Return to the Sass file and create a new custom mixin to define a new custom button called: 'custom-btn' 
    @include extjs-button-ui(
    'custom-btn',

    $border-color: #132A03 #324B22 #324B22 #132A03,
    $border-color-over: #132A03 #324B22 #324B22 #132A03,
    $border-color-focus: #132A03 #324B22 #324B22 #132A03,
    $border-color-pressed: #132A03 #324B22 #324B22 #132A03,
    $border-color-disabled: #132A03 #324B22 #324B22 #132A03,

    $padding: 3px 0,
    $text-padding: null,

    $background-color: #056A2B,
    $background-color-over: #70B840,
    $background-color-focus: #70B840,
    $background-color-pressed: #70B840,
    $background-color-disabled: #70B840,

    $color: #C4D82C,
    $color-over: #000,
    $color-focus: #000,
    $color-pressed: #100,
    $color-disabled: #500,

    $font-size: 18px,
    $font-size-over: 18px,
    $font-size-focus: 18px,
    $font-size-pressed: 18px,
    $font-size-disabled: 18px,

    $font-weight: bold,
    $font-weight-over: bold,
    $font-weight-focus: bold,
    $font-weight-pressed: bold,
    $font-weight-disabled: bold,

    $font-family: sans-serif,
    $font-family-over: sans-serif,
    $font-family-focus: sans-serif,
    $font-family-pressed: sans-serif,
    $font-family-disabled: sans-serif,

    $icon-size: null
);

.x-btn {
    border-width: 5px !important;
}
  9. Open app/view/Viewport.js. Where indicated by the comment, add the following configuration attribute to the button: 
    ui: 'custom-btn',
  10. Save both files.
  11. Preview your application.

    Browse to Labs/Theming/LabCustomUI/MyThemeApp/. If everything is correct your application should look like the picture below:
