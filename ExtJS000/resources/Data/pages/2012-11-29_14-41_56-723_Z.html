Ext.toolbar.Toolbar      |      xtype : 'toolbar'

A toolbar is a container that typically contains buttons. The buttons are styled. Toolbars are almost always used with panels. 

Ext.create('Ext.panel.Panel', {
    title : 'My Panel',
    renderTo : Ext.getBody(),
    bodyPadding : 8,
    layout : 'vbox',
    items : [ {
        xtype : 'button',
        text : 'Button One'
    }],
    dockedItems : [ {
        xtype : 'toolbar',
        dock : 'top',
        items : [ {
            xtype : 'button',
            text : 'Button Alpha'
        }]
    }]
});
