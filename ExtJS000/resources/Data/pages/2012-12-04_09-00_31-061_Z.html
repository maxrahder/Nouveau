VJET is an Eclipse editor plugin that makes developing Ext JS applications faster and easier. It provides stronger JavaScript typing and code completion for Ext JS classes and your subclasses.

VJET gives support for:

  • Ext JS built-in classes (like Ext.panel.Panel)
  • subclasses of Ext JS types
  • your own custom types

Use the Ext JS Type Library (typelib) included in the package. Be sure to select the appropriate framework version type library.

  1. In Eclipse, select File > Import... to open the Import wizard.

  2. Select General > Existing Projects into Workspace and click Next.

  3. Choose the 'Select archive file' option and browse to the downloaded type library project archive file.

  4. IMPORTANT: Check Add project to working sets and click Finish.

    5. Note: This imports the project from the archive into the workspace directory..

    You will notice that ExtJsTL-*** appears as one of your projects in the Script Explorer. Ignore any errors that are reported for ExtJsTL project.
