Use your browser's debugger to inspect the element. Ext JS creates the <div class="x-btn">...</div> dynamically when the button is rendered.

Mouse over the button and see how CSS styles are added to the div. That happens because the component author put low-level DOM events on the div, then ran code to add or remove CSS classes. 

Ext.create('Ext.button.Button',{
    text : 'Button',
    renderTo : Ext.getBody()
});
