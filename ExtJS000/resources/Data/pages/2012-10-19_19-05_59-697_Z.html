Components use an alias type of 'widget'. Other class types will use different alias types: selection, store, proxy, writer, reader, plugin, layout, feature

For example, layout:'vbox' creates an Ext.layout.container.VBox. 

Ext.create('Ext.panel.Panel', {

	layout: 'vbox',

	items : [ {
		xtype: 'button',
		text: 'Button'
	}],
	title: 'My Panel',
	height: 300,
	width: 300,
	bodyPadding: 8,
	renderTo : Ext.getBody(),
});
