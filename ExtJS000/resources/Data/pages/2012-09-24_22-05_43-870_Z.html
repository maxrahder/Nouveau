The items are arranged according to the rules of the container's layout. The layout is specified with the layout property. All containers have an associated layout. 

Ext.create('Ext.container.Container', {
	layout : 'hbox',
	items : [ {
		xtype : 'button',
		text : 'Alpha'
	} , {
		xtype : 'panel',
		text : 'Button',
		bodyPadding: 24,
		items : [ {
			xtype : 'button',
			text : 'Beta'
		} ]
	}],
	renderTo : Ext.getBody(),
	height: 200,
	width: 300
});
