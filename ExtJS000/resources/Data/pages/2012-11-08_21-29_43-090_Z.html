Each item in the chart's series array specifies:

The type config specifies a chart alias: area, bar, column, gauge, line, pie, radar or scatter.

Cartesian and radar charts use xField and yField. Other charts use field.

Cartesian charts also specify which axes are being used: bottom or top and left or right. (Other chart types have a single axis, so there's no need to specify the axis.)
