This property allows you to specify an existing HTML element as the content of a component. It's usually used when combining legacy content with Ext JS content.

This config option is used to take an existing HTML element and place it in the layout element of a new component. It simply moves the specified DOM element after the Component is rendered to use as the content.

index.html
<div id="myContent" class="x-hidden">
My Content, hidden from the page.
</div>
*.js 
Ext.create('Ext.Component', {
  contentEl: 'myContent'
});
