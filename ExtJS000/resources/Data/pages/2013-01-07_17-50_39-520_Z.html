The variable token is embedded in curly braces. It references a property of the passed data object. Use dot notation to get to any part of the data. 

var data = { 
    name: 'Homer Simpson',  
    age: 38,
    wife : {name : 'Marge'} 
};
var c = Ext.create('Ext.Component', {
    tpl : '<h1>{name} is married to {wife.name}</h1>',
    renderTo : Ext.getBody()
});
c.update(data);
