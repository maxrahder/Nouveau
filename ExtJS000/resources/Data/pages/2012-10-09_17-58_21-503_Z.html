Panels also have a tools array. Each element is a config passed for Ext.panel.Tool.

Key configs are:

Running setType() passing a type string changes the icon.

  • type, such as "gear" or "help" which determines the icon.
  • handler
  • tooltip
Ext.create('Ext.panel.Panel', {
    title : 'My Panel',
    renderTo : Ext.getBody(),
    height : 200,
    width : 200,
    tools : [{
        type : 'print'
    }, {
        type : 'gear',
        tooltip : 'Click me',
        handler : function(b){
            alert('Thanks for clicking me.');
        }
    }]
});
