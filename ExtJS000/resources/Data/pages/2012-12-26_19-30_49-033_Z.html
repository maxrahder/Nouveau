Fit layout is used when a container has a single child. The item is automatically sized to take up all the available space. 

Ext.create('Ext.window.Window', {

    layout : 'fit',
    items : [

    {
        xtype : 'component',
        style : {'background-color' : 'red'}
    } ],

    title : 'Window',
    height : 300,
    width : 300,
    bodyPadding: 6,
    x : 1, 
    y : 1,
    autoShow : true

});
