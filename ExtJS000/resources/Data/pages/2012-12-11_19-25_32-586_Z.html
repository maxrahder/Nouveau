Like any OO language, if you need to do further initializations upon creation, you code a constructor. You may not need one at all.

Important: If you create your own class hierarchy, your base class needs a constructor, like the following: 

Ext.define('Mammal', {
    config : {
        name : 'Anonymous',
    },
    constructor : function(config){
        this.initConfig(config);
        this.callParent(config);
        // Do whatever you need here.
    }
});
