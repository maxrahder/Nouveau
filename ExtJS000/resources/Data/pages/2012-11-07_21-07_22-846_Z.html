There are several functions relating to low-level DOM. Some functions return DOM elements, and some return Ext JS DOM wrappers.

  • Ext.getDoc()
  • Ext.getHead()

  • Ext.getGet()
  • Ext.getFly()
  • Ext.getDom()

  • Ext.query() — returns an array of DOM elements
  • Ext.select() — returns an Ext.dom.CompositeElement
