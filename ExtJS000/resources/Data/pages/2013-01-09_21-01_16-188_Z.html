As we discussed previously, default validation occurs whenever a change event is detected. You may, however, wish to defer your validation checking until a form submission is requested or some other event takes place. To allow this to occur, the first step in this process is to disable immediate validation check by setting the validateOnChange field property to false. Once this has been accomplished you can manually invoke validation checks by invoking the following methods:

The following methods allow you to manually invoke data validation:

Ext.form.Field.validate()

Ext.form.Basic.isValid()

Ext.form.Basic.hasInvalidField()

Note: By default, field validation occurs whenever the form's submit() method is invoked. Disabled fields are not validated.
