The ability to dock items is a panel's super power.

dockedItems is an array. Each element in the array must be a component tagged with a dock config, which specifies whether the component is docked to the top, right, bottom or left.

Typically, toolbars are docked but in theory any component can be docked. In this case, there are two items in the dockedItems array: the toolbar at the top, and a button at the bottom. 

Ext.create('Ext.panel.Panel', {

    title : 'My Panel',

    dockedItems : [ {
        xtype : 'toolbar',
        dock : 'top',
        items : [ {
            text : 'Toolbar Button'
        }]
    }, {
        xtype : 'button',
        dock : 'bottom',
        text : 'Bottom Button'
    } ],
    
    width : 300,
    bodyPadding : 16,
    layout : 'vbox',
    items : [ {
        text : 'Center Button',
        xtype : 'button'
    }],

    renderTo : Ext.getBody()
});
