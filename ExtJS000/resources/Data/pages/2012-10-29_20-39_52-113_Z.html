Ext.Array.difference(arrayA, arrayB)

Perform a set difference A-B by subtracting all items in array B from array A.

var o = {a : 13};
var a = [o, 0, 1, 'a', {b: 1}, 'x'];
var b = [o, 1, 'a', {b: 1}, 'y'];
// c = a - b
var c = Ext.Array.difference(a, b);
console.log(c); // alerts [0, {b: 1}, 'x'] - the array with differences
