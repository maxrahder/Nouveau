Introduction

In this lab, you will use Sencha Architect to build an application that shows iTunes music video titles in a grid, thumbnails in a data view, and lets the user play the preview video in a popup window.
Try out http://localhost/SenchaTraining/ExtJS251/LabSolutions/Architect/Tunes/app.html.

The lab lets you use some of the Ext JS features mentioned in the Overview topic. Since we haven't really covered coding in Ext JS very much, we will use Sencha Architect, which will let us quickly create the application without having to get into a lot of programming details.

This is an instructor-led lab. In each step the instructor will explain the goal of the step, then work the step in front of the class. After answering any questions, the class will then work the step.

Please do not work ahead. Save your work frequently.

  1. Application overview

    The data looks like this. The feed is an array of objects, each of which represents a music video. The model will define a field for each music video proprety — the model describes a record. The store is a collection of the records in the feed. The data view and grid are bound to the store.

  2. Initial setup

    Open Sencha Architect.

    On the initial welcome page, under Create New Project, select the latest version of Ext JS.

    After Sencha Architect opens, click the [Save] button on the Architect main toolbar at the top.

    Set the Save Path to [webserver]/SenchaTraining/ExtJS251/Labs/Architect

    Set the Project Name to NepTunes.

    Click [Save].

    Click [Settings] on the Architect main toolbar, and select the Project tab.

    Enter the following URL prefix: http://localhost/SenchaTraining/ExtJS251/Labs/Architect/NepTunes

  3. Set the application name

    In the Project Inspector list on the upper right, select the Application.

    In the Config panel in the lower right, set the name property to NepTunes. You can find the property quickly by typing name in the filter text field near the top of the Config panel. You can also 'Quick Value Set' by typing name:'Tunes' in the filter field itself.

  4. Add a viewport

    In the Toolbox on the left, select a Container | Viewport and drag it to the Canvas.

    In the Design View, click the flyout gear just outside the upper right corner of MyViewport and set layout to vbox.

  5. Add a grid

    In the Toolbox, select a Views | Grid Panel and drag it to the design view. The Project Inspector should look like this:

    IN the center Design Canvas, you'll see a grid with placeholder columns and fake data for prototyping.

    In the Design Canvas, double-click on the title My Grid Panel and change it to NepTunes Grid.

    In the Project Inspector, NepTunes iTunes Grid and in the Config panel's filter box, type itemId to find that property, then set the grid's itemId to neptunesGrid. Note that neptunesGrid now appears in the Project Inspector under MyViewport.

  6. Save and preview the application

    On the Architect main toolbar in the upper le, press [Save] then [Preview]. You should see your application. It has a panel titled NepTunes Grid with placeholder columns but no data. We'll add the data later.

  7. Add a panel and title it NepTunes Music Videos

    In the Toolbox, select a Views | Panel and drag it to the Project Inspector on top of MyViewport. The Project Inspector should look like this:

    In the Design Canvas, double-click on the title My Panel and change it to NepTunes Music Videos.

    In the Design View, click the flyout gear just outside the upper right corner of the panel and check autoscroll and from Select a layout choose fit.

  8. Add a data view in the NepTunes Music Videos panel

    In the Toolbox, select a Views | View and drag it to the body of the NepTunes Music Videos panel. Alternately, you can drag it onto NepTunes Music Videos (under MyViewport) in the Project Inspector. The Project Inspector should look like this:

    Note that a Ext.view.View component is commonly called a data view. It will contain an image of each of the videos.

    In the Project Inspector, select the data view MyView, and in the Config panel set its itemId to neptunesView.

  9. Save and preview the application

    On the Architect main toolbar press [Save] then [Preview]. You should see your application with two panels labeled NepTunes Grid and NepTunes Music Videos.

  10. Set the theme to Neptune

    We want to change the theme of our app from it's 'classic' light blue, to Neptune, which offers a bolder blue.

    At the bottom of the Project Inspector, click on the Library Resource item. Then in the Config panel, set the theme from classic to neptune.

  11. Save and preview the application

    On the Architect main toolbar press [Save] then [Preview] to see the new theme/skin. You can try out the other themes too. We will be discussing theming later in the week.
