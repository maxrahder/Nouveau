• JavaScript has a new operator
  • new used to be used routinely to create JavaScript primitives
  • JSON has made that unnecessary
// The old way
var o = new Object();
o.property = 'my property';
o.items = new Array();
o.items.push(1);
o.items.push(2);
o.items.push(3);

// Using JSON
var o = {property: 'my property', items: [1, 2, 3]};
var d = new Date(); // There is no JSON syntax for date values
