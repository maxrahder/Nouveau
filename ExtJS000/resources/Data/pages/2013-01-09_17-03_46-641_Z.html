Introduction

In this lab, you will use Sencha Architect to define a Template for the List view in your Twitter search mobile application.

Steps

  1. In the Project Inspector, click the Tweets list view.

  2. In the Config filter, type in store.

  3. Choose the store property below and select Tweets from the drop down box.

  4. In the Project Inspector, click the itemTpl that is below the Tweets list view.

  5. In the Design Canvas, click the Code button.

  6. In the code editor, type in (or copy and paste) the following code to display the Twitter fields in the list view:

    7.            
    <img src="{profile_image_url}" align="left" style="margin:5px"/>
{from_user} {text} {created_at}

  7. Save and Preview the project.
