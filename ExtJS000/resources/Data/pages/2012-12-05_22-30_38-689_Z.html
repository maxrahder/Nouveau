...a conspiracy where some Web sites agree to comply with the JSON-P convention.

To wit: the browser sends a parameter, typically called callback. The server then processes the request, and formats the response as a function call, using the callback name.

For example, if send this URL to the iTunes JSONP server
https://itunes.apple.com/us/rss/topmovies/limit=5/json?callback=howdeedo

iTunes responds with
howdeedo({"feed":{"author":{"name":{"label":"iTunes Store"}, "uri":{"label":"

Look at the callback parameter. Whatever you send is used in the response.
