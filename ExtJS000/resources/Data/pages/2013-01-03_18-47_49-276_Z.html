A critical issue when adding an event handler is that you need a reference to the object firing the event.

Doing this directly in the view is easy: you just add a handler as part of the button declaration.* 

tbar : [ {
    text : 'She loves me...',
    itemId : 'sheLovesMe',
    handler : function(button) {
        Ext.Msg.alert(button.getText(), 'Yeah yeah yeah!');
    }
} ]

* With other types of observable objects, you might use the listeners property.
