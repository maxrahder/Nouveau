Aside from the tpl config which allows changing the contents within a components pre-existing html. There is another template config called renderTpl, overriding this property gives us control over the HTML generated by the component as we will be replacing the template it utilizes internally.

Ext.create('Ext.Component', {
    renderTpl: '<ul><li>Item 1</li><li>Item 2</li></ul>',
    renderTo: Ext.getBody()
});
