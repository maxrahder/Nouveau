Introduction

In this short lab, you'll dynamically add tabs to a tab panel.

  1. Review the solution

    Run the solution. The application contains a split button, and an empty tab panel. As you choose from the split button, a new tab is added.

  2. Take a look at the code

    Edit Labs/TabPanel/app/view/Viewport.js. Note how the toolbar is coded: when the user selects an item from the split button, the addPerson() method is run.

  3. Finish the implementation of addPerson()

    The addPerson() method has comments telling you what to do.

    You may want to review the API documentation for the add() method. The add() method is straightforward. You can add an item config exactly like you would add it in the container's items array.

    The add method returns a reference to the newly added component. To make it the current tab, run setActiveTab(), passing the new component.

Conclusion

You can easily add components to any container by passing the config object for the new component.
