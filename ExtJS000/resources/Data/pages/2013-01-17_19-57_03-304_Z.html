To use a locale file, have your server inspect the Accept-Language header in the request* and include a script tag to load the corresponding local file after the Ext JS library has been loaded. 

<!DOCTYPE HTML>
<html>
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="resources/MyApp-all.css"/>
    <script src="all-classes.js"></script>
    <script src="ext-lang-de.js"></script>
</head>
<body></body>
</html>

* You could also specify the locale in a query field.
