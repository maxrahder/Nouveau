Introduction

In this lab, you will use Sencha Architect to define a Controller to enable the Search Field for your Twitter search mobile application.

Steps

  1. In the upper-right corner of the Project Inspector, click [+] and choose Controller.

  2. Select MyController under the Controllers section of the Project Inspector.

  3. In the Config filter, start to type user. This will filter the user named properties.

  4. Select userClassName below and set the value to TwitterSearchController.

  5. In the Config filter, type in actions.

  6. In the Actions property below, click [+] to add a Controller Action.

  7. Choose a target type: Ext.field.Search

  8. Choose an event by name: blur

  9. Click the arrow to select onSearchFieldBlur and add the following code in the editor:

    10. Ext.getStore('Tweets').load({
    params : {
    q : textfield.getValue()
  }
});

  10. Save and Preview.
