You can use Ext.ComponentQuery.query() any time you need to select components. The method returns an array of selected components.

This selects all buttons.

Ext.ComponentQuery.query('button')

Used on the command line, Ext.ComponentQuery.query() is a handy way to select, manipulate and test your own components. 

Ext.ComponentQuery.query('mytabpanel')[0].add({title : 'New Panel'})
alert(Ext.ComponentQuery.query('panel').length);
