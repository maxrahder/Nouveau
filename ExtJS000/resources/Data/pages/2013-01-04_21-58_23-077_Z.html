With the adoption of JSON, and the fact that there is no function overloading in JavaScript, a common way to pass configuration information is via a single object of properties and values. Ext.apply() allow you to apply — or overlay — one set of values with another. 


// Ext.apply() overlays one set of properties onto another
var object = {a: 12, b: 'hi', c: [1, 2, 3]};
Ext.apply(object, {a: 15, d: 99});
console.log(object); // The a and d properties are over-written

// Ext.applyIf() overlays properties if they don't exist
var o = {a: 12, b: 'hi', c: [1, 2, 3]};
Ext.applyIf(o, {a: 15, d: 99});
console.log(o); // Only the d property is over-written
