Panels and buttons are components, so we can apply component-styling to them, such as specifying margins and padding.

This code also uses the defaults config, which panels have because they're a container. 

Ext.create('Ext.panel.Panel', {
	title : 'My Panel',
	renderTo : Ext.getBody(),
	bodyPadding: 8,
	defaults: {
		xtype: 'button',
		margin: 8
	},
	layout : 'vbox',	
	items : [ {
		text : 'Button One'
	}, {
		text : 'Button One'
	} ]
});
