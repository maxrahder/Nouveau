To run the sencha built-in web server (Jetty 8.1.7), use the following command:

sencha fs web -p 80 start -map /path/to/webroot/

While running the web server, Jetty will need the command prompt / terminal Window. So you can open a new command prompt terminal for running other sencha commands.

To stop the webserver, use the following command: (make sure the port is identical to the port you used to start the web server with.

sencha fs web -p 80 stop
