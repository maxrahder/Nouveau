• A little review ... JavaScript objects have properties.
> var obj = { width: 4 };
  undefined
  • We can access properties using dot notation.
> obj.width // returns 4
  4
  • We can also use array-like notation.
> obj["width"] // returns 4
  4
  • We can even add properties on the fly.
> var obj = { width: 4 };
  undefined
> obj.height = 3;
  3
> obj; // returns an object with { width: 4, height: 3 }
  Object
