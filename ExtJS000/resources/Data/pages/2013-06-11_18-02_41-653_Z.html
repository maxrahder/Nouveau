ExtJS251 is a hands-on, comprehensive course providing a more in-depth coverage of the Ext JS 4.2 framework and UI components, including Charting, Grids, the Ext JS 4 Data Model, Forms, Layouts, and Theming. Our ExtJS221 - Essential Ext JS + Architect Live Online Training course is an overview of Ext JS.

ExtJS251 - Fast Track to Ext JS 4.2 Training + Sencha Architect is designed for the developer who needs to know as many facets of Ext JS as can be absorbed in one week. With double the class time of ExtJS221, you'll be able to address all parts of the framework addressed in ExtJS 221 with more depth as well as additional material that the extra time allows for.

Some of the key items covered only in the Fast Track courses are:

  • Charting
  • Data Associations
  • Editing Grids
  • Trees
  • Stateful UI Components
  • Custom Components
  • DOM Manipulation
  • Drag and Drop
  • Theming

Through intensive hands-on labs, you will learn application architecture best practices and build a functioning application with an Ext JS 4 user interface connecting to live data feeds using JSON. You will also learn how to use Sencha Architect 2.2 to rapidly develop a complete Ext JS 4.2 Applications including MVC, Data, and custom components.

Target Audience

Developers who want to become immediately productive with Ext JS
