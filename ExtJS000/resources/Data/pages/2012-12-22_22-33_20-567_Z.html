• In Ext JS, when we often make functions the values of properties in objects. We then call this property a method.
> var obj = { xyz: function() { return 123; } }
  undefined
> obj.xyz; // outputs the contents of obj.xyz, a function
  function () { return 123; }
> obj.xyz(); // outputs the result of calling the function in obj.xyz: 123
  123
  • In the above case, the function in obj.xyz is an anonymous function with no name.
