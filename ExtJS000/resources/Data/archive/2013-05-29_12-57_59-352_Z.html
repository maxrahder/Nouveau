Ext JS also uses Sass mixins. You can use these to create new ui's. UI's can be handy when you want to create a different set of styles within your theme.

For example, you have a blue Neptune theme, but you may want your Ext.toolbar.Toolbar green instead of blue. (See the KitchenSink example)

In your Sass stylesheet you can write the extjs-toolbar-ui mixin as specified in: the API docs. The first argument is the new ui name, for example: green.

This name you can set in your Ext.tool.Toolbar Ext code to point to the new ui: 

Ext.create('Ext.toolbar.Toolbar', {
    renderTo: document.body,
    width   : 400,
    ui: 'green',
    items: [ { text : 'test' }]
});
