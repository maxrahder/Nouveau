1. From Architect's Toolbox, drag a Panel into the Views section of the Project Inspector.

  2. In the Config panel, give this new panel the following property values:

    • title: ITunes Extplorer

    • userClassName: MainPanel

    • layout: border

    • height: 350

    • width: Remove the default width value by clicking on the (x) to the right.

    Note: The app will use the full width of the browser.

  3. Press the [Code] button in the upper right of the Design Panel so that you can look at the code created by Architect.

Save the Application
