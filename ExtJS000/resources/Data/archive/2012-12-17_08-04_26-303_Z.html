Note that in order to use SASS, you must also have RUBY in your path.

If using OSX, you can add the Ruby path by typing the following at a command prompt:
Note that etc directory is hidden.

sudo vi /etc/paths

You will then add the path to the file using the vi editor. Typically the path resembles the following:

/Users/username/.gem/ruby/1.8/bin

The VI Editor has the following commands:

Command Description
*iInsert text before cursor, until is pressed
:wq Save the modified file and quit vi
*u Undo
  • If you are using windows, you can run the installer. You should check the checkbox to make a class path to Ruby.
  • You will need to restart Windows after completing the Ruby install process.
