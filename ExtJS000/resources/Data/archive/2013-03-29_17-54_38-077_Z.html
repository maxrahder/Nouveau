Package Control (and helpful packages)

These allow you to customize the editor. We already had you add package control. View > Show Console allows you to see under the hood.

Recommended packages

We suggested two packages to add functionality to sublime withh our setup instructions.

JsFormat - a formatting aid
Add https://github.com/jdc0589/JsFormat to the URL Area.

SublimeLinter - a tool that alerts you to possible JS errors
Add https://github.com/SublimeLinter/SublimeLinter to the URL Area.

There are many, many more available and we will address some of those below.

ReadMe Please. Allows you to quickly check out the Readme files of the packages you install.

Color Picker. It allows you to use Cmd + Shift + C (Ctrl + Shift + C) to bring up a Color Picker in CSS and immediately assign the color value in your code.

Line Endings helps you to avoid wrong line ending type with code built by various developers.

Side Bar Enhancements allows you to get many additional enhancements to the editor. Worth checking out the Readme file to see what is possible.

Hyperlink Helper allows you to use google or wikipedia to find appropriate link for the page. You can also set up to wrap text in a link.

Emmet is another helpful package. It uses Emmet shortcut notation to quickly build html code. It builds a full shell. You can also use Ctrl + Opt + Right or Left Arrow to navigate insertion points in the HTML. Cmd + Shift + Period or Comma allows you to navigate tag names, attributes and attribute values. Toggle Comment Cmd + Shift + / (Ctrl + Shift + /) allows you to comment front and ending tags. Another helpful feature will only be there if you add a value in the user Preferences. You will add "auto_id_class": true and then you can go to html tag and by typing # you will get an id and by typing . you will get a class addition. More at docs.emmet.io. (Shortcuts may be different on sublime, so check the Readme.) If you go to the

Sublime Package Control website, you can find additional packages by looking in the Community Sublime Packages. (Might want to look at these. SFTP if you use ftp.
PlainTasks allows you to create a list of to do tasks to manage to do projects.
