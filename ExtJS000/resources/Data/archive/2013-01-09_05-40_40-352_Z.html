A proxy tells the app how to get the data.

We will now attach the proxy to the model:

  1. Click on the ITunesRecord model. In the Config panel, find the Proxy property.

  2. Click on the (+) to the right of Proxy.

  3. Select Ajax Proxy.

  4. Select MyAjaxProxy in the Project Inspector.

  5. In the Config panel give this new proxy the following property values:

    • url: data/ITunesSampleData.json

    • reader: Json Reader

    Note: Your code should look like this:

    Ext.define('ITunesExtplorer.model.ITunesRecord', {
    extend: 'Ext.data.Model',

    fields: [
        {
            name: 'name'
        },
        {
            name: 'artist'
        },
        {
            name: 'link'
        },
        {
            name: 'category'
        },
        {
            name: 'price'
        },
        {
            name: 'image'
        },
        {
            name: 'id'
        }
    ],

    proxy: {
        type: 'ajax',
        url: 'data/ITunesSampleData.json',
        reader: {
            type: 'json'
        }
    }
});

Save the Application
