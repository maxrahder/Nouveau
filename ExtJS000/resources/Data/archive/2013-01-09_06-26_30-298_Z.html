Now we can make all the records appear in the list panel on the left. To do this, we will use a full template.

  1. In the Project Inspector, delete the itemTpl by right-clicking on it, and from the menu that appears, select Delete.

  2. Ensure that MyView is selected in the Project Inspector.

  3. In the Config panel, find the tpl property, and click on the (+) on the right.

  4. In the Project Inspector, select tpl.

  5. Select the [Code] button above the Design Canvas/Code View panel (in the upper right corner) so that you can edit the code. Templates are an example of code that can be edited directly in Architect.

  6. In Code View, paste in the following code: 

    <tpl for=".">
    <figure style="display:inline-block;
            margin:5px; width:100px; height:180px;
            vertical-align:top">
        <img src="{image}" width="100">
        <figcaption>{artist}</figcaption>
    </figure>
</tpl>

  7. In the Project Inspector, select MyView

  8. In the Config panel give this view the following property values:

    • store: ITunesStore

    • itemSelector: figure

Note: The item selector identifies which tag in the template marks a separate item. In this case, it is the figure tag.
