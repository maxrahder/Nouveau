<head>
<link rel="stylesheet" type="text/css" href="/extjs4/resources/css/ext-all.css">
<script src="/extjs4/ext-all.js"></script>
<script>
    var message = ':-)';
    Ext.onReady(function() {
        Ext.Msg.alert('Alert', message);
    });
</script>
</head>

"Hey Ext JS, once everything is ready, here's some code I want you to run."

Note that the message variable is outside of the onReady. It would work either way, but the point is that Ext JS code that tries to render components needs to be in the onReady function.
