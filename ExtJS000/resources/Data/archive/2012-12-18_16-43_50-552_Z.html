Introduction

In this lab, you will work with SASS to customize a theme.

Objectives

  • Change the base color of a theme
  • Add custom css styles for the application

Working with Sass

  1. Change the base color of the theme

    Open: packages/YelpTheme/sass/etc/all.scss. Add the following code to this file to change the base color of your theme:

    $base-color: #EBEDCE;
  2. Take over the css styles

    When you open stylesheet.css (created in YelpExtplorer Lab 6 / Templates) you will find some custom css styles: 

    .businesses figure {
    margin: 0;
}
.businesses figcaption {
    text-align: center;
    font-size: 7pt;
}
.businesses .business {
    height: 134px;
    width: 108px;
    margin: 4px;
    float: left;
    border: 4px solid #aaaaaa;
}
.businesses .business img {
    height: 100px;
    width: 100px;
}
.businesses .over {
    border: 4px solid #555555;
}
.businesses .selected {
    border: 4px solid black;
}

    Copy the styles from stylesheet.css. Since these styles are specific for your YelpExtplorer application and not for the theme in general; create a new folder and file in your app directory: YelpExtplorer/sass/src/view/View.scss that uses the styles for the dataview. (xtype: view, that's why the file is called View.scss).

  3. Remove the following link tag from the YelpExtplorer's index.html file. 
    link rel="stylesheet" href="stylesheet.css"
  4. Build the YelpExtplorer to use the theme changes.

    Run the following command from the YelpExtplorer folder.

    sencha app build
  5. Preview the YelpExtplorer in your browser
