When we click on a record on the left, we want to see the details about that record in the form on the right. To set this up, we use a controller.

A controller tells your application to listen for certain events on certain components, and respond to them using a given method.

  1. At the top of the Project Inspector, click on the [+] button, and select Controller.

  2. Select MyController in the Project Inspector.

  3. In the Config panel give this new controller the following property value:

    • userClassName: ITunesController

  4. To set up the event listener,

    • Click on the (+) to the right of Actions.

    • From the menu, select Controller Action.

    • For the Target Type, select the class of our list component, Ext.view.View.

    • Choose the select event.

  5. Now we write the method.

    • In the Project Inspector, click on select onDataViewSelect.

    • Select the [Code] button.

    • Enter the following code in the editor: 

var basicForm = Ext.ComponentQuery.query('form')[0].getForm();
basicForm.loadRecord(record);
