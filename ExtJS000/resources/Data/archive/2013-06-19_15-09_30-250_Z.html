Introduction

In this lab, you will generate a new theme for the YelpExtplorer app.

Objectives

  • Review requirements for theming an application
  • Generate a new theme
  • Inspect files of a new theme
  • Extend from an existing theme.
  • Modify an application to use a new theme

Generate a theme

Note: This lab requires that Sencha Cmd and Ruby are correctly installed. (See Setup Instructions in the Appendix)

  1. Navigate in the console to the application folder (SenchaTraining/ExtJS251/Labs/YelpExtplorer).
  2. Generate the theme with Sencha Cmd:

    Note: If the YelpTheme already exist, delete it before performing this step. 

    sencha generate theme YelpTheme
  3. Preview the new folder structure:

    Browse with Windows Explorer or Finder to the SenchaTraining/ExtJS251/Labs/packages/YelpTheme/ folder, notice the package folder structure.

  4. Extend from the gray theme:

    Open SenchaTraining/ExtJS251/Labs/packages/YelpTheme/package.json and alter the file so that it will now extend from ext-theme-gray

        "extend": "ext-theme-gray"
  5. Change the YelpExtplorer's sencha.cfg file to point to a different theme:

    Open the hidden file /YelpExtplorer/.sencha/app/sencha.cfg and switch the theme

    app.theme=YelpTheme
  6. Build the YelpExtplorer to use the new theme:

    Note: If you've run a build using a different theme, clean the build directory by running the sencha ant clean command before performing this step.

    Run the following command from the YelpExtplorer folder.

    sencha app build

    7. By the end of this lab, you should see a new generated YelpExtplorer YelpTheme theme which extends from the gray theme.

    A custom theme can extend from the classic Ext JS theme or from the new Neptune theme.

    If you are an advanced theme developer, you could also extend from the neutral or base theme. Remember that if you start there, you will be adding all of the styling and that will require effort and planning.
